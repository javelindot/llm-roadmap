# 深度学习进阶

> 深入理解深度学习核心技术

## 目录简介

本目录涵盖深度学习的基础理论和核心架构，从神经网络基础到CNN、RNN、Transformer等经典模型。

## 章节结构

### 第一章：神经网络基础
- 神经元与激活函数
- 前向传播与反向传播
- 损失函数与优化器
- 梯度下降与正则化

### 第二章：CNN 卷积神经网络
- 卷积层与池化层
- 经典CNN架构：LeNet、AlexNet、VGG、ResNet
- 图像分类与目标检测

### 第三章：RNN 循环神经网络
- RNN基础与LSTM
- GRU与序列建模
- 文本生成与序列标注

### 第四章：Attention 机制
- 注意力机制原理
- Self-Attention与Multi-Head Attention
- Transformer架构详解

## 学习建议

1. **理论与实践结合**：理解原理后，使用PyTorch/TensorFlow实现
2. **从简单到复杂**：先掌握基础神经网络，再学习复杂架构
3. **关注前沿**：深度学习领域发展迅速，保持学习最新论文

## 前置知识

- Python编程基础
- 线性代数与微积分
- 机器学习基础
