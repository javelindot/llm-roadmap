面试常考的 7 大题型，每类至少刷 5 题：

| 题型 | 代表题 | 频率 |
|------|--------|------|
| 数组 / 双指针 | 三数之和、接雨水 | ⭐⭐⭐⭐⭐ |
| 哈希 / 滑窗 | 最长无重复子串 | ⭐⭐⭐⭐⭐ |
| BFS / DFS | 岛屿数量、二叉树路径 | ⭐⭐⭐⭐ |
| 动态规划 | 最长递增子序列、背包 | ⭐⭐⭐⭐ |
| 堆 / 优先队列 | Top-K 频次、合并 K 链表 | ⭐⭐⭐ |
| 回溯 | 子集、全排列、N 皇后 | ⭐⭐⭐ |
| 图 / 并查集 | 拓扑排序、最短路 | ⭐⭐ |
