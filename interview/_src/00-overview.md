2025–2026 大模型岗位主要分三大方向：

| 方向 | 核心技能 | 面试重点 |
|------|----------|----------|
| 算法 / 训练 | 预训练、SFT、RLHF、对齐 | 八股 + 论文 + 调参经验 |
| 应用 / Agent | Prompt、RAG、Agent、工具集成 | 项目深挖 + 系统设计 |
| 基建 / 推理 | vLLM、TensorRT、量化、分布式 | 性能优化 + 工程实战 |

:::callout 🎯
**建议**：先选定一个主方向深入，再用 20% 时间补另一方向。多方向「全都浅尝」反而最难突围。
:::
