# 模型训练的时候梯度如何传播？

## 答案

模型训练的梯度通过反向传播和链式法则，从损失函数沿计算图逆向传到每个可训练参数。

### 1. 前向传播

例如两层网络：

```text
z1 = W1*x + b1
h1 = activation(z1)
logits = W2*h1 + b2
loss = CE(logits, label)
```

前向阶段保存反向计算需要的中间激活。

### 2. 反向传播

从：

```text
dLoss / dLogits
```

开始，利用链式法则：

```text
dLoss/dW2 = dLoss/dLogits * dLogits/dW2
dLoss/dh1 = dLoss/dLogits * dLogits/dh1
dLoss/dW1 = dLoss/dh1 * dh1/dz1 * dz1/dW1
```

自动微分框架记录算子及其依赖，调用 `loss.backward()` 后按拓扑逆序执行各算子的 backward。

### 3. Transformer 中的传播

梯度会经过：

- LM Head；
- FFN；
- Residual；
- Norm；
- Attention 输出投影；
- Q/K/V 投影；
- Embedding。

若 embedding 和 LM Head 共享权重，同一个参数会收到两条路径累积的梯度。

### 4. 梯度累积与清零

默认情况下，PyTorch 梯度会累加：

```python
optimizer.zero_grad()
loss.backward()
optimizer.step()
```

梯度累积时：

```python
loss = loss / accumulation_steps
loss.backward()
```

累积若干 micro-batch 后再 `step`，从而模拟更大的 global batch。

### 5. 分布式训练

Data Parallel/DDP 中，每张卡计算本地梯度，再通过 All-Reduce 求和或平均，保证各副本使用一致梯度更新。ZeRO-2/3 常使用 Reduce-Scatter，让每张卡只保留自己的梯度分片。

### 6. 常见问题

- 梯度消失：深层链式乘积过小；
- 梯度爆炸：梯度范数急剧增大；
- 梯度为零：死 ReLU、错误 mask、参数冻结；
- NaN/Inf：学习率过高、低精度溢出、数据异常；
- 没有梯度：使用了 `detach`、`no_grad` 或参数没进 optimizer。

排查可记录：

```text
各层 grad norm
全局 grad norm
参数更新比例
NaN/Inf
激活分布
```

### 7. 面试总结

梯度传播就是从 loss 出发，通过计算图和链式法则逆向求导；梯度先累积到参数，再由优化器更新。完整回答还应覆盖残差通路、梯度累积、分布式归约以及消失、爆炸和 NaN 的排查。
