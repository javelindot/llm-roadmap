# 🔥🔥 梯度消失和梯度爆炸怎么解决？

## 答案

梯度消失和爆炸是多层链式求导中 Jacobian 连乘导致梯度指数缩小或放大。应先监控每层梯度范数和首个异常位置，再针对结构、初始化和优化处理。

### 缓解梯度消失

- 使用 ReLU、GELU、SiLU 等非饱和或平滑激活，避免深层 sigmoid/tanh 长期处于饱和区；
- 残差连接提供恒等梯度通道；
- Transformer 使用 Pre-LN/Pre-RMSNorm；
- 合理初始化：ReLU 常用 He，tanh 常用 Xavier；
- RNN 使用 LSTM/GRU 门控或截断 BPTT；
- 必要时使用辅助损失、跳连和较浅网络。

### 抑制梯度爆炸

- Global Gradient Norm Clipping；
- 降低学习率并增加 warmup；
- 合理初始化和残差缩放；
- 增大有效 batch，减少梯度噪声；
- BF16 或动态 loss scaling，关键累加保留 FP32；
- 检查异常数据、mask、除零、log/softmax 数值稳定；
- 分布式训练确认计算的是全局而非单卡局部范数。

### 工程排查

记录 loss spike 对应 batch、每层激活与梯度、参数更新/参数范数比、NaN/Inf。先在单卡小数据 FP32/BF16 复现，再逐步打开混合精度和并行。

### 面试总结

梯度消失靠残差、归一化、激活和初始化改善传播；梯度爆炸靠裁剪、学习率、数值精度和数据治理控制。真正定位要看逐层梯度与异常 batch，不能只机械调学习率。
