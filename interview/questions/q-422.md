# 🔥🔥 知道哪些损失函数

## 答案

**一句话**：损失函数要按任务选择，分类常用交叉熵，回归常用 MSE/MAE，排序常用 pairwise/listwise loss，大模型训练常用 next-token cross entropy，对齐会用偏好损失或 RL 目标。

### 1. 常见损失函数

- MSE：回归任务，惩罚大误差更明显。
- MAE：回归任务，对异常值更鲁棒。
- BCE：二分类或多标签分类。
- Cross Entropy：多分类、语言模型 next-token 预测。
- Hinge Loss：SVM、间隔分类。
- Contrastive / Triplet / InfoNCE：对比学习和表征学习。
- Pairwise Ranking Loss：排序任务。
- KL Loss：分布对齐、蒸馏、RLHF 约束。
- DPO Loss：偏好对齐。

### 2. 结合跨境支付项目怎么说（面试口径）

> 在跨境支付项目里，不同任务用不同损失。比如工单意图分类可以用交叉熵，判断用户是付款失败、退款咨询还是 KYC 问题；风控评分如果是回归风险分，可以用 MSE 或 MAE；如果做相似工单检索，可以用对比学习损失；如果做模型客服话术对齐，可以用 SFT 的 next-token loss 或 DPO 偏好损失。
>
> 我不会孤立背损失函数，而是先看任务目标、标签形式和业务指标。支付场景里误判成本不一样，比如把高风险交易判成低风险比反过来更严重，这时还要考虑样本权重和阈值策略。

**钩子**：可以引到“分类、回归、排序、对比学习、SFT/DPO 在业务里的对应关系”。
