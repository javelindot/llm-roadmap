# BERT系列训练流程与各种变体

## 答案

BERT 是双向 Transformer Encoder。原始预训练输入由 token、segment 和可学习绝对 position embedding 相加，核心目标是 MLM 和 NSP。

### 原始训练流程

语料经 WordPiece 分词，构造单句或句对并加入 `[CLS]`、`[SEP]`。MLM 随机选择约 15% token，其中一部分替换为 `[MASK]`、随机词或保持不变，让模型恢复原 token；NSP 判断句子 B 是否真实接在 A 后。预训练后增加任务头，端到端微调分类、序列标注、问答等任务。

### 代表性变体

- RoBERTa：更多数据和训练步数、动态 masking，移除 NSP。
- ALBERT：因式分解 embedding 参数、跨层参数共享，并使用 sentence order prediction。
- ELECTRA：生成器替换 token，判别器判断每个 token 是否被替换，提高样本效率。
- DeBERTa：内容与位置解耦的注意力，并增强相对位置建模。
- SpanBERT：mask 连续 span，适合跨度和关系任务。
- DistilBERT/TinyBERT：知识蒸馏，降低参数和延迟。
- ERNIE 等：融合实体、短语或知识增强目标。

### 选型思路

BERT 类模型适合理解、分类和 token-level 任务；生成任务通常使用 encoder-decoder 或 decoder-only。比较变体时要控制预训练数据和模型规模，不能把所有收益归因于单一结构。

下游微调需关注学习率、最大长度、类别不均衡、层冻结和多次随机种子。小数据时可使用 layer-wise learning-rate decay、early stopping 或 PEFT，并检查 train/test 文本泄漏。
