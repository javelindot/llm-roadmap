# LoRA 训练和全参训练的优劣

## 答案

LoRA 与全参数训练的核心权衡是资源和模块化能力对模型适配容量。

### LoRA 优点

- 只训练低秩增量，梯度与优化器状态少，训练显存和 checkpoint 大幅降低。
- 一个基础模型可挂载多个小适配器，便于多租户、版本管理和快速回滚。
- 小数据场景下参数约束有时能降低过拟合和通用能力遗忘。
- 单适配器可合并到原权重中，推理不必增加额外层。

### LoRA 局限

- 低秩假设和 target modules 限制适配容量，领域迁移很大或需要重塑底层知识时可能不如全参。
- 基础权重仍需加载，激活显存也不会因可训练参数减少而消失。
- 动态多 LoRA 服务会增加调度与访存复杂度。
- rank、alpha、目标层和数据质量仍需调参，参数少不代表不会过拟合。

### 全参数训练

优点是所有层和方向都可更新，容量最大，数据充足且领域偏移明显时通常有更高上限，也便于得到一个独立、无需适配器管理的最终模型。

缺点是梯度和 Adam 状态昂贵，需要更强的分布式训练；每个任务都保存完整模型，存储和部署成本高；小数据或学习率不当更容易灾难性遗忘。

### 如何选择

先用 LoRA 建立质量/成本基线。若关键指标明显受容量限制，可增加 rank 和目标模块，再考虑部分层解冻或全参训练。应在相同数据、global batch 和训练 token 下比较效果、吞吐、峰值显存、总成本及通用能力退化，而不是只比较可训练参数比例。
