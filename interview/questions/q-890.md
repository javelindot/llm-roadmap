# bert的原理

## 答案

BERT 的核心是用双向 Transformer Encoder 和 Masked Language Modeling 学习上下文化表示。

### 1. 输入表示

\[
E=E_{\rm token}+E_{\rm position}+E_{\rm segment}
\]

`[CLS]` 用于句级任务，`[SEP]` 分隔句子，Segment Embedding 区分 A/B。

### 2. 双向 Encoder

每个 token 可关注左右全部非 padding token。每层由 Multi-Head Self-Attention、FFN、残差和 LayerNorm 构成。与自回归 Decoder 不同，它不使用 causal mask。

### 3. 预训练

MLM 将部分 token 替换为 `[MASK]`、随机词或保持原词，让模型恢复原 token：

\[
\mathcal L_{\rm MLM}
=-\sum_{t\in M}\log P(x_t|x_{\setminus M})
\]

原始 BERT 还使用 NSP。后续 RoBERTa 移除 NSP、动态 mask 并扩大数据，说明训练配方很关键。

### 4. 微调

- 分类：`[CLS]` 或池化接线性层；
- NER：每 token 分类；
- 问答：预测答案起止；
- rerank：query/document 拼接后打分。

所有参数可端到端更新，也可用 Adapter/LoRA。检索 embedding 通常需专门对比学习。

### 5. 局限

MLM 与下游生成不一致，不适合长文本自回归生成；固定绝对位置和二次 Attention 限制长度；`[MASK]` 在真实输入中不存在，存在预训练/微调差异。

### 面试总结

BERT 用 MLM 训练双向 Encoder，从左右上下文恢复被遮挡 token，再通过轻量任务头适配分类、抽取与重排。优势是理解，局限是生成和长上下文。
