# 残差连接如何缓解梯度消失和爆炸问题？

## 答案

残差块可以写成：

```text
y = x + F(x)
```

反向传播时：

```text
dy/dx = I + dF/dx
```

即使 `dF/dx` 很小，梯度仍可以通过恒等分支中的 `I` 直接向前层传播，因此比纯粹经过很多非线性层的链式乘积更不容易消失。

### 1. 缓解梯度消失

普通深层网络：

```text
gradient = J_n * J_(n-1) * ... * J_1
```

若多个 Jacobian 的范数小于 1，梯度会指数衰减。

残差网络每层是：

```text
I + J_F
```

提供更短、更直接的梯度路径，也让无用层可以学习 `F(x)≈0`，整体接近恒等映射，缓解网络加深后的优化退化。

### 2. 对梯度爆炸的作用边界

残差连接并不天然保证不爆炸。若每层残差分支过大，连续的：

```text
I + J_F
```

仍可能使梯度增长。还需要：

- 合理初始化；
- LayerNorm/RMSNorm；
- Pre-Norm；
- 残差缩放；
- 较小学习率和 warmup；
- gradient clipping；
- bf16 或数值稳定实现。

### 3. Transformer 中的残差

典型 Pre-Norm Transformer：

```text
x = x + Attention(Norm(x))
x = x + FFN(Norm(x))
```

Pre-Norm 让残差主干更接近恒等路径，通常比深层 Post-Norm 更容易训练。超深模型还可能使用 residual scaling、DeepNorm 等策略。

### 4. 面试总结

残差连接通过恒等 shortcut 为信息和梯度提供直达路径，核心导数是 `I + dF/dx`，主要缓解梯度消失和深层网络退化。它不能单独解决梯度爆炸，仍要配合归一化、初始化、学习率和裁剪。
