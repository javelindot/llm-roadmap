# 我问啥样的领导行呢？

## 答案

好领导不能只用“会赚钱”概括，应同时看方向判断、交付能力、用人方式和职业操守。

### 可观察的标准

- 能把公司目标转成清晰、稳定且可衡量的团队目标；
- 理解业务和技术，能在关键取舍上作出决策；
- 能争取资源，也敢停止低价值项目；
- 分工、绩效和晋升标准透明，功劳归团队、责任不下甩；
- 愿意培养成员，提供反馈和有挑战性的 ownership；
- 遇到事故先保护用户和解决问题，再做无责复盘；
- 能在合规和长期价值内创造结果，而非透支团队或风险套利。

尽调时少听抽象评价，多问事实：核心成员平均任期、过去一年上线内容、1:1 与 code review 机制、绩效分布、最近一次失败如何处理。

跨境支付领导还应尊重牌照、制裁、隐私和资金安全边界。短期收入不能成为绕过合规控制的理由。

### 面试总结

好领导既能选对方向并拿到结果，也能建立清晰机制、培养成员和承担责任。长期看，团队持续交付能力比一次短期增长更能说明领导质量。
