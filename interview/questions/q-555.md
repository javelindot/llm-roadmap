# Self-Attention vs Cross-Attention 的区别?

## 答案

Self-Attention 和 Cross-Attention 使用相同的注意力公式，核心区别是 Q/K/V 来自哪里。

### 1. Self-Attention

Q、K、V 都来自同一个序列：

```text
Q = XW_Q
K = XW_K
V = XW_V
```

作用是让序列内部不同位置互相交换信息。

- Encoder Self-Attention 通常是双向的。
- Decoder Self-Attention 使用 Causal Mask，只能看当前位置之前的 Token。

### 2. Cross-Attention

Q 来自当前正在更新的序列，K/V 来自另一个序列或模态：

```text
Q = X_target W_Q
K = X_source W_K
V = X_source W_V
```

例如机器翻译中，Decoder Query 读取 Encoder 的源语言表示；多模态模型中，文本 Query 可以读取图像特征。

### 3. 对比

| 维度 | Self-Attention | Cross-Attention |
|------|----------------|-----------------|
| Q 来源 | 当前序列 | 目标序列 |
| K/V 来源 | 当前序列 | 源序列/另一模态 |
| 主要作用 | 序列内部建模 | 两个表示空间交互 |
| 注意力矩阵 | `n × n` | `n × m` |
| 复杂度 | `O(n²d)` | `O(nmd)` |

### 4. 常见架构

- BERT：Encoder Self-Attention。
- GPT/LLaMA：Causal Self-Attention。
- 原始 Transformer：Encoder Self-Attention + Decoder Self-Attention + Cross-Attention。
- 视觉语言模型：图像—文本 Cross-Attention 或投影后统一 Self-Attention。

**面试总结**：Self-Attention 回答“同一序列内部该看谁”，Cross-Attention 回答“当前序列该从另一个序列读取什么”。
