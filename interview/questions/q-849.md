# 实现Top-p采样算法

## 答案

Top-p（Nucleus Sampling）先按概率从大到小排序，选择累计概率达到 \(p\) 的最小 token 集合，在集合内重新归一化后采样。候选数会随分布动态变化。

```python
import math
import random


def top_p_sample(logits, p=0.9, temperature=1.0, rng=None):
    if not logits:
        raise ValueError("logits must not be empty")
    if not 0 < p <= 1:
        raise ValueError("p must be in (0, 1]")
    if temperature <= 0:
        raise ValueError("temperature must be positive")

    rng = rng or random
    scaled = [value / temperature for value in logits]
    max_logit = max(scaled)
    weights = [math.exp(value - max_logit) for value in scaled]
    total = sum(weights)
    probabilities = [value / total for value in weights]

    ranked = sorted(
        enumerate(probabilities),
        key=lambda item: item[1],
        reverse=True,
    )

    nucleus = []
    cumulative = 0.0
    for token_id, probability in ranked:
        nucleus.append((token_id, probability))
        cumulative += probability
        if cumulative >= p:
            break

    nucleus_total = sum(probability for _, probability in nucleus)
    threshold = rng.random() * nucleus_total
    cumulative = 0.0
    for token_id, probability in nucleus:
        cumulative += probability
        if cumulative >= threshold:
            return token_id

    return nucleus[-1][0]
```

实现要先减最大 logit 防止指数溢出。工程框架通常使用排序/选择 kernel，并可先应用 repetition penalty、temperature、top-k，再做 top-p。\(p=1\) 退化为全分布采样；temperature 越低，分布越尖。

### 面试总结

Top-p 保留累计概率达到阈值的最小高概率集合，再重新归一化采样。关键边界是数值稳定、\(p\) 范围、至少保留一个 token，以及解码过滤顺序。
