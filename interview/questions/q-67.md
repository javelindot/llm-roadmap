# LLMs 采用的注意力机制变体

## 答案

**一句话**：从 **MHA → MQA → GQA → MLA**，变体核心都在解决一件事——**怎么省 KV Cache 同时不丢质量**。

### 1. MHA（Multi-Head Attention，原始）
- 每个 head 有自己的 Q、K、V
- 头数 = h，每头维度 = d/h
- KV Cache 大小 = 2 × h × seq × d/h = 2 × seq × d
- 训练效果最好但推理显存爆

### 2. MQA（Multi-Query Attention）
- **所有 Q 头共享 1 个 K、1 个 V**
- KV Cache 缩小 h 倍
- 推理快、显存省
- 质量略降，模型不稳
- 代表：PaLM、Falcon

### 3. GQA（Grouped-Query Attention）⭐ 主流
- Q 分 g 组，每组共享 1 个 K、V（g 一般 = 8）
- 是 MHA 和 MQA 的中间态
- KV Cache 比 MHA 小 h/g 倍
- 质量几乎和 MHA 持平
- 代表：**Llama-2/3、Qwen-2/3、Mistral、Gemma**

### 4. MLA（Multi-head Latent Attention）
- DeepSeek-V2/V3 提出
- 把 K/V **低秩压缩**到一个 latent 向量
- 推理时只缓存 latent，再 up-project
- KV Cache 比 GQA 还小，质量优于 MHA
- 代表：DeepSeek-V2、V3、R1

### 5. 其他变体（场景化）
| 变体 | 思路 | 用例 |
|------|------|------|
| **Sliding Window Attention** | 只看最近 W token | Mistral |
| **Sparse Attention** | 固定+全局+滑窗 | Longformer |
| **Linear Attention** | softmax 近似线性化 | Performer / RWKV |
| **State Space** | SSM 替代 Attention | Mamba / Mamba-2 |
| **Hybrid** | Attn + SSM 混合 | Jamba |

### 6. 选型对比表
| 变体 | KV Cache | 质量 | 训练难度 | 业界采用 |
|------|---------|------|---------|---------|
| MHA | 1x | 100% | 简单 | 早期 |
| MQA | 1/h | 95% | 中 | 少 |
| **GQA** | 1/(h/g) | **99%** | 简单 | **主流** |
| **MLA** | <1/h | **101%** | 较难 | DeepSeek |

### 7. 工程经验
- 新项目选 base 优先 GQA（Llama-3 / Qwen-3 / Mistral）
- 自训模型推理优化无脑 GQA
- 算力极紧、追极致显存 → MLA
- 长上下文 → GQA + Sliding Window 或 SSM Hybrid

### 8. 一句话总结
**GQA 是当前事实标准**（Llama-3 / Qwen-3），**MLA 是 DeepSeek 的下一代**，工业 95% 场景选 GQA 就对了。

## 结合项目谈（口语版 · 约 40 秒）

> 我们跨境支付的训练 base 选型时**KV Cache 占用是个硬指标**，因为客服对话和风控调查都吃长上下文，KV Cache 一爆推理成本就翻倍。
>
> 我们最早用的是 Qwen-7B（MHA），32k 上下文 KV Cache 占 14G 左右，一张 A100 80G 只能跑 4 路并发。后来升级 Qwen2.5-7B（**GQA**，g=4），同样 32k 上下文 KV Cache 只占 3.5G，**并发能拉到 16 路**，单机吞吐翻 4 倍——这是切 GQA 实打实的收益。
>
> 我们也调研过 **DeepSeek-V2 的 MLA**，KV Cache 比 GQA 还小 60%，理论上更适合长上下文场景。但 DeepSeek-V2 部署生态成熟度不如 Qwen，业务侧定制比较麻烦，最后没上生产。等 DeepSeek-V3 / R1 周边稳了之后我们会重新评估。
>
> 还有一个观察是 **GQA 训练几乎没坑**——直接换 base 就行，不像 MQA 有时不稳，也不像 MLA 训练实现复杂。
>
> 我的总结：**GQA 是当前工程默认选项**，KV Cache 省 4x 质量不掉，是免费午餐；MLA 是更激进的下一代，长期看好但要等生态。
