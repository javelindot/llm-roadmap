# 介绍一下Bert

## 答案

BERT 是 Encoder-only 的双向 Transformer 预训练模型，通过 Masked Language Modeling 学习上下文化 token 表示，适合理解、分类和抽取任务。

### 1. 输入

每个位置由三部分相加：

\[
E=E_{\rm token}+E_{\rm position}+E_{\rm segment}
\]

`[CLS]` 用于句级表示，`[SEP]` 分隔句子，Segment Embedding 区分句子 A/B。

### 2. Encoder

每层包含 Multi-Head Self-Attention、残差、LayerNorm 和两层 GELU FFN。原始 BERT 使用 Post-Norm。

- BERT Base：12 层、768 hidden、12 heads，约 110M；
- BERT Large：24 层、1024 hidden、16 heads，约 340M。

### 3. 预训练目标

- MLM：随机 mask 部分 token，根据左右上下文预测原 token；
- NSP：判断两句是否相邻。RoBERTa 等后续工作表明移除 NSP 并优化训练配方也能更好。

### 4. 下游任务

- 分类：`[CLS]` 或 pooling 接分类头；
- NER：每个 token 接分类头/CRF；
- 抽取式问答：预测 start/end；
- rerank：query 与 document 拼接做 Cross-Encoder；
- embedding：需经过对比学习微调，原始 `[CLS]` 不一定适合向量检索。

### 5. 局限

BERT 不能自然做自回归长文本生成，MLM 预训练与生成不同；绝对位置和 \(O(n^2)\) Attention 限制长度；预训练语料知识也会过时。

### 面试总结

BERT 用双向 Transformer Encoder 和 MLM 学上下文表示，适合分类、NER、问答与重排。它不是生成模型，原始向量也不等于优质检索 embedding。
