# 知道Lora中的scale吗？

## 答案

LoRA 中常说的 scale 是低秩更新的缩放系数：

\[
y=Wx+sBAx,\qquad s=\frac{\alpha}{r}
\]

其中 \(r\) 是 rank，\(\alpha\) 是 `lora_alpha`。它控制 LoRA 分支相对基座权重的影响。

### 作用

- scale 太小：更新影响弱，可能欠拟合；
- scale 太大：更新过强，训练不稳定、遗忘或格式漂移；
- \(r\) 改变时，使用 \(\alpha/r\) 可限制更新尺度随 rank 线性增大；
- rsLoRA 使用 \(\alpha/\sqrt r\)，避免较大 rank 时梯度和有效更新过弱。

不要把 scale 与 optimizer learning rate 混为一谈。学习率控制每步如何更新 \(A,B\)，scale 控制前向中 \(\Delta W\) 的整体幅度，两者会共同影响训练。

推理框架还可能提供 adapter multiplier，在不重训时临时调整 LoRA 强度；超过训练附近范围容易出现不可预测行为，必须验证。

### 面试总结

标准 LoRA scale 通常是 \(\alpha/r\)，用于控制低秩增量的幅度；rsLoRA 常用 \(\alpha/\sqrt r\)。设置要与 rank、学习率和目标模块一起通过消融选择。
