# 手撕： transformers encoder

## 答案

下面是一个简化的 Pre-Norm Transformer Encoder Layer，包含多头自注意力、前馈网络、残差、LayerNorm 和 padding mask：

```python
import torch
import torch.nn as nn


class TransformerEncoderLayer(nn.Module):
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        assert d_model % num_heads == 0
        self.num_heads = num_heads
        self.head_dim = d_model // num_heads
        self.scale = self.head_dim ** -0.5

        self.qkv = nn.Linear(d_model, 3 * d_model)
        self.out_proj = nn.Linear(d_model, d_model)
        self.ffn = nn.Sequential(
            nn.Linear(d_model, d_ff),
            nn.GELU(),
            nn.Dropout(dropout),
            nn.Linear(d_ff, d_model),
        )
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout = nn.Dropout(dropout)

    def attention(self, x, padding_mask=None):
        batch, seq_len, d_model = x.shape
        qkv = self.qkv(x).reshape(
            batch, seq_len, 3, self.num_heads, self.head_dim
        )
        q, k, v = qkv.unbind(dim=2)
        q = q.transpose(1, 2)
        k = k.transpose(1, 2)
        v = v.transpose(1, 2)

        scores = torch.matmul(q, k.transpose(-2, -1)) * self.scale
        if padding_mask is not None:
            # padding_mask: [batch, seq_len], True 表示 padding
            scores = scores.masked_fill(
                padding_mask[:, None, None, :], float("-inf")
            )

        weights = torch.softmax(scores, dim=-1)
        weights = self.dropout(weights)
        context = torch.matmul(weights, v)
        context = context.transpose(1, 2).contiguous()
        context = context.reshape(batch, seq_len, d_model)
        return self.out_proj(context)

    def forward(self, x, padding_mask=None):
        x = x + self.dropout(self.attention(self.norm1(x), padding_mask))
        x = x + self.dropout(self.ffn(self.norm2(x)))
        return x
```

### 关键点

输入形状是 `[batch, seq_len, d_model]`。Q/K/V 拆成多个头后为 `[batch, heads, seq_len, head_dim]`，注意力分数为 `[batch, heads, seq_len, seq_len]`。Encoder 使用双向注意力，通常不加 causal mask，但需要屏蔽 padding key。

完整 Encoder 还要在输入加入位置编码，堆叠多层，并在需要时增加最终归一化和任务头。生产实现可使用 `torch.nn.functional.scaled_dot_product_attention` 或 FlashAttention，以减少显存并避免手写 softmax 的数值与性能问题。
