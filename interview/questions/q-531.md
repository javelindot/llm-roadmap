# 多模态模型怎么训练？

## 答案

多模态模型训练通常不是一步端到端完成，而是分阶段把“单模态能力、跨模态对齐、指令遵循和偏好”逐步建立起来。

### 1. 单模态预训练

- 文本侧使用预训练 LLM。
- 图像侧使用 ViT/CLIP Encoder。
- 音频侧使用声学 Encoder。

复用成熟单模态模型可以显著降低数据和算力成本。

### 2. 跨模态对齐

冻结或部分冻结两端，训练 Projector、Adapter、Q-Former 或 Cross-Attention，使视觉/音频特征能被语言模型读取。

常见目标：

- 图文对比学习；
- 图文匹配；
- Caption 生成；
- Masked Modality Modeling。

### 3. 多模态指令微调

使用“多模态输入 + 指令 + 回答”数据训练模型完成：

- 图像问答；
- OCR / 文档理解；
- 图表推理；
- 视频理解；
- 多轮对话。

这一步重点是任务覆盖、指令多样性和答案质量。

### 4. 联合微调或解冻

当连接模块已稳定后，可逐步解冻 LLM、模态 Encoder 的部分层，做联合训练以提升细粒度能力。通常使用较小学习率，防止破坏单模态能力。

### 5. 偏好与安全对齐

使用人工偏好、DPO/RLHF 或规则数据优化帮助性、拒答、视觉忠实度和安全性。

### 6. 数据与工程难点

- 图文对是否真正对应；
- 不同模态 Token 数量不平衡；
- 高分辨率与长视频显存开销；
- 模型根据语言先验忽略视觉证据；
- 多数据集采样比例和灾难性遗忘。

**面试总结**：典型路线是“单模态预训练 → 连接层对齐 → 多模态指令微调 → 按需联合解冻 → 偏好与安全对齐”。
