# post norm和pre norm的区别？

## 答案

两者区别是 LayerNorm 放在残差子层之前还是之后。

### Post-Norm

\[
x_{l+1}=\operatorname{LN}(x_l+F(x_l))
\]

这是原始 Transformer 的形式。每层输出都被归一化，但梯度沿残差路径仍要经过 LayerNorm 的 Jacobian。网络很深时更难优化，通常更依赖 warmup、初始化和残差缩放。

### Pre-Norm

\[
x_{l+1}=x_l+F(\operatorname{LN}(x_l))
\]

残差主干有直接恒等梯度：

\[
\frac{\partial x_{l+1}}{\partial x_l}
=I+\frac{\partial F}{\partial x_l}
\]

因此深层模型训练通常更稳定，现代 LLM 多使用 Pre-LN 或 Pre-RMSNorm，并在整个网络末尾再加一次 Norm。

### 取舍

- Pre-Norm：优化稳定、易扩深，但不同层可能更接近恒等更新，需要合理的残差尺度；
- Post-Norm：理论上每层输出规范化更强，某些设置下最终效果可能好，但深层训练更困难；
- DeepNorm、Sandwich Norm、QK-Norm 等是在稳定性和表达能力之间进一步改造，不能简单归为“Pre 永远更好”。

跨境支付模型微调通常沿用基座 Norm 结构，不应为了领域适配随意改动，否则预训练权重分布会失配。

### 面试总结

Post-Norm 是 `LN(x+F(x))`，Pre-Norm 是 `x+F(LN(x))`。Pre-Norm 的恒等残差梯度使深层训练更稳定，所以现代 LLM 更常见；Post-Norm 可能有表示优势，但需要更谨慎的优化设计。
