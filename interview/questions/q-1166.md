# 打分样本空间分布与精排不一致的解决方案

## 答案

打分模型的训练样本分布与线上精排候选分布不一致，本质上是 covariate shift、sample selection bias 或策略变化。模型在训练集上学到的决策边界，到了精排真正面对的“高相似、高难度候选”上会失效。

### 1. 先定位分布差异来自哪里

常见原因包括：

- 训练样本从全量曝光随机抽取，线上精排只接收召回和粗排筛过的困难候选；
- 负样本主要是随机负例，而线上候选大多同类、相似且容易混淆；
- 上游召回模型、配额或过滤规则升级，但精排训练数据仍来自旧策略；
- 训练按 item 均匀采样，线上按热门度或用户活跃度分布；
- 训练使用最终曝光样本，缺少“进入精排但未曝光”的候选；
- 位置、曝光和点击偏差被当成真实偏好；
- 特征版本、缺失率或实时新鲜度与线上不一致。

应比较训练集、精排输入和最终曝光三个阶段的特征分布、类别比例、候选来源、分数区间及人群分桶，并计算 PSI、KL/JS divergence、KS 等漂移指标。更重要的是沿 trace 还原每个样本由哪版召回和粗排策略产生。

### 2. 用线上精排候选重建训练空间

最直接的方案是记录精排入口的完整候选及特征快照：

```text
request_id、user/context、item_id、recall_source、
pre_rank_score、rank_position、feature_version、policy_version
```

训练时优先从真实精排候选池采样，而不是只从全库随机抽负例。可以组合：

- 已曝光点击或转化的正样本；
- 已曝光未点击样本；
- 进入精排但被模型淘汰的 hard negative；
- 最新召回或粗排模型产生的近邻困难样本；
- 少量随机样本保持全局覆盖；
- 新品、长尾和高风险场景的挑战样本。

未曝光候选没有直接行为标签，不能简单全部标为负例。可用于 pairwise/listwise 蒸馏、旧模型软标签或受控探索数据。

### 3. 做重要性加权和去偏

若能估计训练分布 `p_train(x)` 与目标分布 `p_target(x)`，可以使用重要性权重：

```text
w(x) = p_target(x) / p_train(x)
L = Σ w(x) × loss(x, y)
```

实际可训练一个 domain classifier 区分“训练样本”和“线上精排样本”，再由分类概率估计 density ratio。权重要裁剪并归一化，避免少量样本产生极大梯度。

对于曝光策略偏差，可记录 propensity，使用 IPS/SNIPS、Doubly Robust 等方法校正。若历史日志缺少随机探索或 propensity，反事实估计会不可靠，此时应先通过小流量随机化收集可识别数据。

### 4. 让模型适应最新上游分布

- 上游召回或粗排大改时，使用新策略 shadow 产出的候选回放；
- 周期性挖掘当前模型 Top-K 的 false positive 作为 hard negative；
- 使用 teacher 对精排候选做 listwise 打分，蒸馏到线上模型；
- 将召回来源、粗排分数、候选 rank 和策略版本作为特征；
- 对不同召回域使用 domain-specific BN、adapter 或专家；
- 增量训练时混入稳定历史样本，避免只追逐短期分布。

训练与上线要绑定 `feature、recall、pre-rank、rank-model` 版本。否则模型指标变化时无法判断是数据漂移还是模型退化。

### 5. 校准和评测必须贴近线上

测试集应至少包含：

- 与当前线上候选生成策略同分布的 on-policy 集；
- 按时间后切分的未来集；
- 新品、长尾、冷用户和不同召回来源分桶；
- 同类困难负样本和上游策略变更挑战集。

除 AUC、GAUC、LogLoss 和 NDCG 外，还要看分桶校准、Top-K precision、模型分数分布和最终业务指标。模型在随机测试集上 AUC 很高，但在精排入口的困难候选上 NDCG 下降，仍然不能上线。

### 6. 跨境支付中的对应问题

支付通道路由或商户产品排序也会出现相同偏差。例如训练数据来自历史已执行通道，但线上候选经过国家、币种、金额、牌照和实时健康度过滤，模型实际面对的是一组高度相似的可行通道。

应记录“当时所有合法候选”，而不只记录最终执行通道，并区分：

- 因硬规则不可用；
- 被排序模型淘汰；
- 被成本或时效策略选择；
- 实际执行后成功或失败。

不能把因合规不允许而未执行的通道当作普通负样本，也不能用交易完成后才知道的信息训练曝光时打分。最终评测需要同时观察成功率、成本、到账时效、覆盖率、风险与稳定性。
