# 先做了两道题，一道easy，一道mid，都是链表。

## 答案

原题没有给出两道链表题的完整题干，因此不能唯一确定答案。面试复盘时最好补全输入、输出、边界条件和题目链接。链表 easy/mid 高频考点可以用下面的方法快速识别。

### 1. 常见题型和解法

- 反转链表：三个指针 `prev/current/next`；
- 合并有序链表：dummy node + 双指针；
- 判断环：快慢指针；
- 找环入口：快慢指针相遇后，一个指针回到头部；
- 找中点：快慢指针；
- 删除倒数第 N 个：快指针先走 N 步；
- 回文链表：找中点 + 反转后半段 + 比较；
- 重排链表：找中点 + 反转后半段 + 交替合并；
- 链表排序：归并排序。

### 2. 通用节点定义

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

### 3. 高频模板

反转链表：

```python
def reverse(head):
    prev = None
    cur = head

    while cur:
        nxt = cur.next
        cur.next = prev
        prev = cur
        cur = nxt

    return prev
```

找中点：

```python
def middle(head):
    slow = fast = head

    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next

    return slow
```

合并两个有序链表：

```python
def merge(a, b):
    dummy = ListNode()
    tail = dummy

    while a and b:
        if a.val <= b.val:
            tail.next, a = a, a.next
        else:
            tail.next, b = b, b.next
        tail = tail.next

    tail.next = a or b
    return dummy.next
```

### 4. 面试时怎么表达

写代码前先确认：

- 空链表怎么处理；
- 节点值是否重复；
- 能否修改原链表；
- 是否要求 O(1) 额外空间；
- 返回头节点还是原地修改；
- 偶数长度时中点取左还是取右。

### 5. 面试总结

这类复盘题信息不足，不能假装知道原题。真正要准备的是链表的四个基础模块：dummy node、快慢指针、原地反转、归并。大多数 easy/mid 链表题都是这些模块的组合。
