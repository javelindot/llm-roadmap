# ViT的结构和原理

## 答案

**一句话**：ViT 把图片切成 patch，把每个 patch 当成 token，再用 Transformer 做全局建模。

### 结构

1. 图片切成固定大小 patch。
2. 每个 patch 展平并线性投影成 embedding。
3. 加位置编码。
4. 加 CLS token。
5. 输入 Transformer Encoder。
6. 用 CLS 或 pooling 做分类。

### 特点

ViT 擅长全局关系建模，但通常需要较多数据；CNN 有更强局部归纳偏置，小数据下更稳。

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付里，ViT 可以用于证件、银行回单、KYC 材料分类和文档图像理解。比如把扫描件切成 patch 后，模型能关注不同区域之间的关系，如公司名、证件号、有效期。
>
> 但对于 OCR 字段抽取，通常还要结合布局模型、OCR 和规则校验，不能只靠 ViT 分类。

**钩子**：可以引到“patch token、位置编码、文档图像、KYC 材料识别”。
