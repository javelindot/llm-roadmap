# 手撕transformer模型代码，介绍模型架构，介绍交叉熵损失函数，介绍L1，L2正则化有什么区别，写下完整损失函数公式，写下transformer所有矩阵转换的数学公式，如何理解需要除根号d

## 答案

Transformer 由注意力、前馈网络、残差连接、归一化和位置编码组成。原始架构是 Encoder-Decoder；BERT 使用 Encoder-only，GPT 类模型使用带 causal mask 的 Decoder-only。

### 1. 矩阵与形状

设输入 `X` 形状为 `[B, N, d_model]`，head 数为 `h`，`d_k = d_model / h`。

**线性投影**

```text
Q = X * W_Q
K = X * W_K
V = X * W_V

W_Q, W_K, W_V: [d_model, h * d_k]
Q, K, V:       [B, N, h * d_k]
```

变形并转置：

```text
[B, N, h * d_k] → [B, N, h, d_k] → [B, h, N, d_k]
```

**Scaled Dot-Product Attention**

```text
S = Q * K^T / sqrt(d_k) + Mask
A = softmax(S, dim=-1)
H = A * V

S, A: [B, h, N_query, N_key]
H:    [B, h, N_query, d_k]
```

拼接多头：

```text
Concat(H) → [B, N_query, h * d_k]
O = Concat(H) * W_O
```

**前馈网络**

```text
FFN(X) = activation(X * W_1 + b_1) * W_2 + b_2
W_1: [d_model, d_ff]
W_2: [d_ff, d_model]
```

Pre-Norm Encoder layer 常写成：

```text
X_1 = X + MHA(LN(X))
X_2 = X_1 + FFN(LN(X_1))
```

Decoder self-attention 使用 causal mask。Encoder-Decoder Cross-Attention 中：

```text
Q = DecoderState * W_Q
K = EncoderOutput * W_K
V = EncoderOutput * W_V
```

最终 hidden state 投影到词表：

```text
Logits = Z * W_vocab + b_vocab
P = softmax(Logits)
```

### 2. 为什么除以 `sqrt(d_k)`

若 Q、K 每个维度近似独立、均值 0、方差 1，点积 `q · k` 是 `d_k` 项之和，方差约为 `d_k`。维度增大时 logits 绝对值会变大，softmax 进入饱和区，梯度变小。

除以 `sqrt(d_k)` 后，logit 方差回到约 1 的量级，使训练更稳定。除的是每个 head 的维度 `d_k`，不是整个序列长度。

### 3. 可手写的 PyTorch 核心

```python
import math
import torch
from torch import nn


class MultiHeadAttention(nn.Module):
    def __init__(self, d_model: int, num_heads: int):
        super().__init__()
        if d_model % num_heads != 0:
            raise ValueError("d_model must be divisible by num_heads")
        self.num_heads = num_heads
        self.head_dim = d_model // num_heads
        self.q_proj = nn.Linear(d_model, d_model)
        self.k_proj = nn.Linear(d_model, d_model)
        self.v_proj = nn.Linear(d_model, d_model)
        self.o_proj = nn.Linear(d_model, d_model)

    def split_heads(self, x: torch.Tensor) -> torch.Tensor:
        batch, seq_len, _ = x.shape
        x = x.view(batch, seq_len, self.num_heads, self.head_dim)
        return x.transpose(1, 2)

    def forward(self, query, key, value, mask=None):
        q = self.split_heads(self.q_proj(query))
        k = self.split_heads(self.k_proj(key))
        v = self.split_heads(self.v_proj(value))

        scores = q @ k.transpose(-2, -1)
        scores = scores / math.sqrt(self.head_dim)
        if mask is not None:
            scores = scores.masked_fill(~mask.to(torch.bool), -1e9)

        weights = torch.softmax(scores, dim=-1)
        context = weights @ v
        batch, _, query_len, _ = context.shape
        context = context.transpose(1, 2).contiguous()
        context = context.view(batch, query_len, -1)
        return self.o_proj(context)


class FeedForward(nn.Module):
    def __init__(self, d_model: int, d_ff: int):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(d_model, d_ff),
            nn.GELU(),
            nn.Linear(d_ff, d_model),
        )

    def forward(self, x):
        return self.net(x)


class EncoderLayer(nn.Module):
    def __init__(self, d_model: int, num_heads: int, d_ff: int):
        super().__init__()
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.attn = MultiHeadAttention(d_model, num_heads)
        self.ffn = FeedForward(d_model, d_ff)

    def forward(self, x, mask=None):
        h = self.norm1(x)
        x = x + self.attn(h, h, h, mask)
        x = x + self.ffn(self.norm2(x))
        return x
```

完整实现还应加入 embedding、位置编码、dropout、多层堆叠、padding/causal mask、输出 head 和初始化。生产环境优先使用框架的 fused attention，而不是手写版本。

### 4. 交叉熵损失

对第 `t` 个 token 的真实标签 `y_t`：

```text
CE = -Σ_t log softmax(logits_t)[y_t]
```

实现时通常设置 padding label 为 `ignore_index`，只对目标 token 计算平均 loss。SFT 中还可屏蔽 Prompt token，只监督回答部分。

### 5. L1 与 L2 正则

```text
L1 = Σ_i |theta_i|
L2 = Σ_i theta_i^2
```

- L1 在 0 点不可导，容易得到稀疏参数，适合特征选择；
- L2 对大权重惩罚更强，使权重平滑收缩，通常不会精确变成 0；
- AdamW 的 decoupled weight decay 与直接把 L2 加入 loss 在自适应优化器下不完全等价；
- Transformer 训练中一般不对 bias 和 Norm 参数应用 weight decay。

### 6. 完整损失

同时使用 L1、L2 时可以写为：

```text
L_total =
    L_cross_entropy
    + lambda_1 * Σ_i |theta_i|
    + (lambda_2 / 2) * Σ_i theta_i^2
```

实际大模型训练更常见的是 cross entropy + AdamW weight decay，并结合 label smoothing、dropout 或任务特定辅助 loss；不一定同时使用显式 L1 和 L2。

### 7. 边界与稳定性

- Mask 的形状必须可广播到 `[B, h, N_query, N_key]`；
- 全屏蔽行不能直接 softmax，否则可能产生 NaN；
- `view` 前注意内存连续性；
- 混合精度下 softmax 和 mask 值要避免溢出；
- Cross-Attention 的 Query 长度可与 Key/Value 长度不同；
- 训练时能并行计算全部位置，Decoder 推理仍需逐 token 生成并使用 KV Cache。
