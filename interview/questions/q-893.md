# 基于值函数的和基于策略的有哪些模型？

## 答案

强化学习方法可分为基于值函数、基于策略和 Actor-Critic，现代算法经常是组合而非绝对二分。

### 1. 基于值函数

学习状态价值 \(V(s)\) 或动作价值 \(Q(s,a)\)，策略通常选价值最大的动作。

- Dynamic Programming：Value Iteration、Policy Iteration；
- Monte Carlo Control；
- TD、SARSA；
- Q-Learning；
- DQN 及 Double DQN、Dueling DQN、Rainbow；
- 离线 RL 中 CQL、IQL 等也高度依赖价值学习。

适合离散动作；连续动作需要求 \(\arg\max_a Q(s,a)\)，通常较困难。

### 2. 基于策略

直接参数化：

\[
\pi_\theta(a|s)
\]

并优化期望回报。

- REINFORCE；
- Vanilla Policy Gradient；
- TRPO；
- PPO；
- 大模型中的 GRPO/DAPO 也属于策略优化家族。

它适合连续/高维动作和随机策略，但梯度方差较高、样本效率可能较低。

### 3. Actor-Critic

Actor 表示策略，Critic 学 \(V/Q\) 作为 baseline 或训练信号：

- A2C/A3C；
- DDPG；
- TD3；
- SAC；
- PPO 通常也配 value critic；
- IMPALA。

它结合策略表达能力与价值估计的低方差，但同时训练两个模型会带来偏差和稳定性问题。

### 4. 大模型对应

经典 PPO-RLHF 是 Actor-Critic；DPO 是离线偏好优化，不是传统 value-based RL；GRPO 用同 prompt 组内 reward 代替独立 critic。

### 面试总结

值函数方法先学动作好坏再选动作，代表 Q-Learning/DQN；策略方法直接优化策略，代表 REINFORCE/PPO；Actor-Critic 用价值函数辅助策略，代表 A2C、SAC、PPO。选型取决于动作空间、采样成本和稳定性。
