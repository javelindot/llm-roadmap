# 手撕位置编码和MHA+力扣完美矩形变体

## 答案

这道题包含三个独立部分：位置编码、Multi-Head Attention 和完美矩形判定。实现时要先确认是否允许调用框架自带 Attention；以下给出核心手写版本。

### 1. Sinusoidal 位置编码

原始 Transformer 的位置编码为：

```text
PE(pos, 2i)   = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
```

PyTorch 实现：

```python
import math
import torch
from torch import nn


class SinusoidalPositionEncoding(nn.Module):
    def __init__(self, d_model: int, max_len: int = 4096):
        super().__init__()
        if d_model <= 0:
            raise ValueError("d_model must be positive")

        position = torch.arange(max_len, dtype=torch.float32).unsqueeze(1)
        div_term = torch.exp(
            torch.arange(0, d_model, 2, dtype=torch.float32)
            * (-math.log(10000.0) / d_model)
        )

        pe = torch.zeros(max_len, d_model)
        pe[:, 0::2] = torch.sin(position * div_term)
        if d_model > 1:
            pe[:, 1::2] = torch.cos(
                position * div_term[: pe[:, 1::2].shape[1]]
            )

        self.register_buffer("pe", pe.unsqueeze(0), persistent=False)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        # x: [batch, seq_len, d_model]
        if x.size(1) > self.pe.size(1):
            raise ValueError("sequence exceeds max_len")
        return x + self.pe[:, : x.size(1)].to(dtype=x.dtype)
```

`register_buffer` 使位置编码随模型迁移设备，但不参与梯度更新。现代 LLM 更常用 RoPE，不过手写题一般先考 Sinusoidal。

### 2. Multi-Head Attention

```python
class MultiHeadAttention(nn.Module):
    def __init__(self, d_model: int, num_heads: int, dropout: float = 0.0):
        super().__init__()
        if d_model % num_heads != 0:
            raise ValueError("d_model must be divisible by num_heads")

        self.num_heads = num_heads
        self.head_dim = d_model // num_heads
        self.q_proj = nn.Linear(d_model, d_model)
        self.k_proj = nn.Linear(d_model, d_model)
        self.v_proj = nn.Linear(d_model, d_model)
        self.out_proj = nn.Linear(d_model, d_model)
        self.dropout = nn.Dropout(dropout)

    def _split_heads(self, x: torch.Tensor) -> torch.Tensor:
        batch, seq_len, _ = x.shape
        return (
            x.view(batch, seq_len, self.num_heads, self.head_dim)
            .transpose(1, 2)
        )

    def forward(
        self,
        query: torch.Tensor,
        key: torch.Tensor,
        value: torch.Tensor,
        mask: torch.Tensor | None = None,
    ) -> tuple[torch.Tensor, torch.Tensor]:
        q = self._split_heads(self.q_proj(query))
        k = self._split_heads(self.k_proj(key))
        v = self._split_heads(self.v_proj(value))

        scores = q @ k.transpose(-2, -1) / math.sqrt(self.head_dim)
        if mask is not None:
            # mask 应可广播到 [batch, heads, query_len, key_len]；
            # True 表示可见，False 表示屏蔽。
            scores = scores.masked_fill(~mask.to(torch.bool), float("-inf"))

        weights = torch.softmax(scores, dim=-1)
        weights = self.dropout(weights)
        context = weights @ v

        batch, _, query_len, _ = context.shape
        context = (
            context.transpose(1, 2)
            .contiguous()
            .view(batch, query_len, self.num_heads * self.head_dim)
        )
        return self.out_proj(context), weights
```

Self-Attention 传入相同的 `query、key、value`；Cross-Attention 的 Query 来自 Decoder，Key/Value 来自 Encoder。生产代码还要处理全屏蔽行、KV Cache、混合精度和高效 Attention kernel。

### 3. 完美矩形

多个轴对齐小矩形恰好覆盖一个大矩形，需要同时满足：

1. 小矩形面积之和等于外接矩形面积；
2. 所有内部顶点出现偶数次并被抵消；
3. 最终只剩外接矩形四个角。

```python
def is_rectangle_cover(rectangles: list[list[int]]) -> bool:
    if not rectangles:
        return False

    min_x = min_y = float("inf")
    max_x = max_y = float("-inf")
    area = 0
    corners: set[tuple[int, int]] = set()

    for x1, y1, x2, y2 in rectangles:
        if x1 >= x2 or y1 >= y2:
            return False

        min_x = min(min_x, x1)
        min_y = min(min_y, y1)
        max_x = max(max_x, x2)
        max_y = max(max_y, y2)
        area += (x2 - x1) * (y2 - y1)

        for point in ((x1, y1), (x1, y2), (x2, y1), (x2, y2)):
            if point in corners:
                corners.remove(point)
            else:
                corners.add(point)

    expected_corners = {
        (min_x, min_y),
        (min_x, max_y),
        (max_x, min_y),
        (max_x, max_y),
    }
    bounding_area = (max_x - min_x) * (max_y - min_y)
    return area == bounding_area and corners == expected_corners
```

时间复杂度为 `O(n)`，额外空间最坏 `O(n)`。面积相等只能排除总量差异，不能单独排除“局部重叠和另一处空洞面积恰好相等”；角点奇偶性与面积必须同时检查。

若变体要求找出重叠区域、支持旋转矩形或在线动态加入矩形，该方法不再直接适用，需要扫描线、线段树或更具体的数据结构。
