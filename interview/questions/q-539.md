# 7.红黑树特点

## 答案

红黑树是一种自平衡二叉搜索树，通过节点颜色和旋转约束树高，使查找、插入、删除都保持 `O(log n)`。

### 1. 五条性质

1. 每个节点是红色或黑色。
2. 根节点是黑色。
3. 所有空叶子 NIL 节点视为黑色。
4. 红色节点的子节点必须是黑色，不能连续出现两个红节点。
5. 从任意节点到其所有后代 NIL 节点的路径，包含相同数量的黑节点。

### 2. 为什么能够平衡

由于红节点不能连续，且每条路径黑节点数量相同，最长路径最多约为最短路径的两倍，因此树高上界是 `O(log n)`。

### 3. 插入与删除

- 插入新节点通常先设为红色，减少对黑高的影响。
- 若违反约束，通过重新着色和左/右旋修复。
- 删除黑节点可能破坏黑高，修复过程比插入更复杂。

### 4. 与 AVL 树比较

| 维度 | 红黑树 | AVL |
|------|--------|-----|
| 平衡程度 | 较宽松 | 更严格 |
| 查询 | 稳定 `O(log n)` | 通常略快 |
| 插入删除 | 旋转相对少 | 调整更频繁 |
| 适合场景 | 读写都频繁 | 查询占绝对多数 |

### 5. 应用

- C++ `std::map` / `std::set` 的常见实现。
- Java `TreeMap`。
- 内核调度、定时器等需要有序动态集合的场景。

**面试总结**：红黑树牺牲一点严格平衡，换取更低的更新成本，同时保证最坏 `O(log n)`。
