# 如何确保分层的准确性，用大模型进行分层为什么没有尝试更大的模型？

## 答案

这里的“分层”可能指标签层级、用户/内容分层或大模型路由。回答时应先确认定义，然后从 taxonomy、标注、模型和评测四层保证准确性。

### 如何保证分层准确

- 先定义互斥性、覆盖性和优先级，给出每层正例、反例与边界案例。
- 多人标注并统计一致率；冲突样本由专家仲裁。
- 将粗层到细层拆成层级分类，父层置信度不足时不进入子层，或允许多标签。
- 使用规则、轻量分类器和 LLM 组合；高置信自动通过，低置信进入复核。
- 通过 structured output 限制合法层级，并验证父子关系。
- 建立按层级、类别和长尾分桶的 precision、recall、macro-F1 与混淆矩阵。

线上持续采样边界和分布漂移样本，校准阈值并版本化 taxonomy。

### 为什么没有使用更大模型

不能只回答“成本高”，应说明经过了质量与成本实验。更大模型的潜在收益要与延迟、调用价格、吞吐、隐私、可部署性和稳定性比较。如果当前模型在高频层级已经达到目标，误差主要来自标签定义或数据歧义，换更大模型不会解决根因。

合理方案是选择代表性困难集对多个尺寸做离线评测，画出质量、延迟和成本 Pareto 曲线。可用小模型处理高置信样本，大模型只复核低置信或高价值样本。若没有做过更大模型实验，应诚实说明资源限制，并给出计划，而不是宣称更大模型一定无收益。
