# 其他归一化的方式还了解哪些

## 答案

归一化方法的区别主要在统计哪些维度、是否依赖 batch，以及是否只做尺度归一。

### 常见方法

- BatchNorm：每通道跨 batch 和空间维统计，CNN 中常用；训练与推理统计不同，小 batch 不稳定。
- LayerNorm：对单样本每个 token 的隐藏维归一，和 batch 无关，Transformer 常用。
- RMSNorm：只按均方根缩放，不减均值，LLaMA 等模型使用。
- GroupNorm：通道分组后在组内归一，适合检测/分割的小 batch CNN。
- InstanceNorm：每样本每通道在空间维归一，风格迁移常用，可能去除实例对比信息。
- SyncBatchNorm：多卡同步 BN 统计量，增加通信换取更大有效 batch。

### 其他变体

- WeightNorm：重参数化权重的长度与方向，不直接归一化激活。
- SpectralNorm：约束权重谱范数，常用于 GAN 判别器稳定性。
- L2 normalization：将 embedding 投影到单位球，检索和对比学习常用。
- Batch Renorm、Ghost BN、Switchable Norm 等针对 batch 或多统计量问题。

### 选型

Transformer 序列长度和 batch 动态，通常选择 LayerNorm/RMSNorm；CNN 大 batch 优先 BN，小 batch可用 GN。归一化位置也重要，Pre-Norm 和 Post-Norm 的梯度路径不同。

归一化不是免费的：BN 有同步和 train/eval 差异，LN/RMSNorm 在 decode 中也有访存成本。应结合任务、batch、硬件和精度压测。
