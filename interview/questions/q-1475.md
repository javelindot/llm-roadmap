# 冒泡排序和选择排序的时间复杂度？

## 答案

冒泡排序和选择排序的平均、最坏时间复杂度都是 O(n²)，空间复杂度都是 O(1)，但最好情况、稳定性和交换次数不同。

### 1. 冒泡排序

相邻元素不断比较和交换，每轮把当前最大元素“冒泡”到末尾。

```python
def bubble_sort(nums):
    n = len(nums)

    for end in range(n - 1, 0, -1):
        swapped = False

        for i in range(end):
            if nums[i] > nums[i + 1]:
                nums[i], nums[i + 1] = nums[i + 1], nums[i]
                swapped = True

        if not swapped:
            break
```

复杂度：

```text
最好：O(n)，需要 swapped 提前退出
平均：O(n²)
最坏：O(n²)
空间：O(1)
```

使用严格 `>` 交换时，冒泡排序是稳定排序。

### 2. 选择排序

每轮从未排序区间找到最小值，交换到前面：

```python
def selection_sort(nums):
    n = len(nums)

    for i in range(n):
        min_index = i

        for j in range(i + 1, n):
            if nums[j] < nums[min_index]:
                min_index = j

        nums[i], nums[min_index] = nums[min_index], nums[i]
```

复杂度：

```text
最好：O(n²)
平均：O(n²)
最坏：O(n²)
空间：O(1)
```

标准选择排序不稳定，因为远距离交换可能改变相同元素的相对顺序。

### 3. 对比

| 维度 | 冒泡排序 | 选择排序 |
|---|---|---|
| 最好时间 | O(n)，可提前退出 | O(n²) |
| 平均/最坏 | O(n²) | O(n²) |
| 稳定性 | 稳定 | 通常不稳定 |
| 交换次数 | 最坏 O(n²) | O(n) |
| 比较次数 | O(n²) | O(n²) |

### 4. 面试总结

两者都不是大数据排序的优先选择。冒泡可利用已有有序性且稳定；选择排序比较次数固定，但交换次数更少。
