# L1和L2的区别

## 答案

L1 和 L2 都通过在原损失上增加参数惩罚限制模型复杂度，但得到的参数形态不同。

### 1. 公式

```text
L1：Loss + λ Σ |w_i|
L2：Loss + λ Σ w_i²
```

### 2. 核心区别

| 维度 | L1 | L2 |
|------|----|----|
| 参数结果 | 容易产生精确的 0 | 参数整体变小但通常不为 0 |
| 作用 | 稀疏化、特征选择 | 平滑收缩、防止权重过大 |
| 零点可导性 | 不可导，使用次梯度 | 可导 |
| 对共线特征 | 可能保留其中少数 | 倾向共同缩小 |
| 异常大权重惩罚 | 线性 | 二次，更强 |

### 3. 为什么 L1 产生稀疏

L1 约束区域有尖角，优化解更容易落在坐标轴上，使部分参数精确为 0。Lasso 就利用这一点做特征选择。

### 4. L2 的直观解释

L2 对大权重惩罚更强，使模型依赖多个较小权重而不是少数极大权重。在线性模型中称 Ridge，在神经网络中常以 Weight Decay 使用。

### 5. 概率解释

- L1 对应 Laplace 先验。
- L2 对应 Gaussian 先验。

### 6. 实践注意

- 特征应先做尺度标准化，否则惩罚不可比。
- `λ` 太大会欠拟合，太小则效果有限。
- Elastic Net 结合 L1 和 L2，兼顾稀疏与稳定。
- AdamW 的 Decoupled Weight Decay 与直接把 L2 加入 Loss 在自适应优化器下并不完全等价。

**面试总结**：L1 适合想要稀疏和特征选择，L2 适合平滑缩小参数并提升稳定性。
