# 求二叉树中两个节点的最近公共祖先

## 答案

对普通二叉树，递归定义是：若当前节点为空或等于 `p/q`，直接返回；左右子树都找到目标时，当前节点就是最近公共祖先；否则返回非空的一侧。

```java
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode(int val) {
        this.val = val;
    }
}

class Solution {
    public TreeNode lowestCommonAncestor(
            TreeNode root, TreeNode p, TreeNode q) {
        if (root == null || root == p || root == q) {
            return root;
        }

        TreeNode left = lowestCommonAncestor(root.left, p, q);
        TreeNode right = lowestCommonAncestor(root.right, p, q);

        if (left != null && right != null) {
            return root;
        }
        return left != null ? left : right;
    }
}
```

这里按节点引用判断，而不是按 `val`，因为树中值可能重复。标准题通常假设 `p`、`q` 都存在；若不保证存在，需要额外返回找到的节点数，确认两者都出现后再给 LCA。

每个节点最多访问一次，时间复杂度 \(O(n)\)，递归栈为 \(O(h)\)，退化链表时最坏 \(O(n)\)。若是二叉搜索树，可利用大小关系降到 \(O(h)\)。

### 面试总结

后序遍历分别在左右子树找目标；左右都非空时当前节点是 LCA，否则答案在非空侧。复杂度 \(O(n)\)，并要说明节点不存在与值重复的边界。
