# DAPO 改进了哪些方面？

## 答案

DAPO 是 2025 年公开的 Decoupled Clip and Dynamic sAmpling Policy Optimization，基于无独立 critic 的组内相对策略优化，针对长推理 RL 的熵塌缩、无效样本和长度偏差提出四项关键改进。

### 1. Clip-Higher

标准 PPO/GRPO 常使用对称 clip。DAPO 将下界和上界解耦，例如下界 0.2、上界 0.28，允许低概率 token 有更大的上调空间，缓解策略熵过快下降并促进探索。

### 2. Dynamic Sampling

如果同一 prompt 的所有 rollout 全对或全错，组内奖励方差为零，advantage 无有效学习信号。DAPO 动态过滤这些样本，并继续采样，直到 batch 中包含有区分度的 prompt。

### 3. Token-Level Policy Gradient Loss

常见实现先对每条 response 的 token loss 求平均，再对样本平均，会让不同长度回答权重近似相同。DAPO 在全局 token 维做归一化，使长推理轨迹中的有效 token 获得更合理的训练权重。

### 4. Overlong Reward Shaping

超过最大长度直接给零奖励会产生突变，且模型在截断前不知道将越界。DAPO 在接近长度上限时逐步施加 soft penalty，并对截断回答惩罚，减少异常超长和 reward noise。

### 5. 效果与边界

[DAPO 论文](https://arxiv.org/abs/2503.14476)与[官方代码](https://github.com/BytedTsinghua-SIA/DAPO)公开了基于 Qwen2.5-32B 和 DAPO-Math-17K 的训练方案，并报告 AIME 2024 达到 50 分。该结论来自数学可验证奖励任务，不能直接外推到主观偏好或真实业务 Agent。

DAPO 配置还去除了 KL penalty，但这不是所有任务都应照搬。开放领域仍需监控策略漂移、reward hacking 与通用能力退化。

### 面试总结

DAPO 的四个核心改进是 Clip-Higher、Dynamic Sampling、Token-Level PG Loss 和 Overlong Reward Shaping，分别解决探索不足、零梯度样本、长度归一化偏差和截断奖励噪声。
