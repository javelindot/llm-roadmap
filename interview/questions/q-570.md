# 6.FlashAttention v1 和 v2 的主要改进点分别是什么？

## 答案

**一句话**：FlashAttention v1 解决了 attention 的 IO 瓶颈，v2 进一步优化并行度、减少非矩阵乘操作，让 GPU 利用率更高。

### v1

- 分块计算 QKV。
- 不显式存完整 attention 矩阵。
- 使用 online softmax。
- 减少 HBM 读写和显存占用。

### v2

- 更好的并行切分策略。
- 减少非 matmul 操作。
- 优化 warp/block 级并行。
- 提高 GPU 利用率，训练和推理更快。

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付长文档问答里，比如通道规则和 KYC 文档，FlashAttention 能降低长上下文模型的计算和显存压力。但我会把它当底层优化，业务侧仍然通过 RAG 控制输入长度。
>
> 也就是说，FlashAttention 让长上下文更可用，RAG 让上下文更有用。

**钩子**：可以引到“IO-aware、online softmax、v2 并行优化、长上下文业务落地”。
