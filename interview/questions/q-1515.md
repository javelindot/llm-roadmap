# 写一下 ppo 算法的损失函数和 GAE 优势函数。主要还是讲明白

## 答案

PPO 用 clipped surrogate objective 限制新策略不要一次偏离旧策略过大；GAE 用多步 TD residual 的指数加权估计优势，在偏差和方差之间折中。

### 1. PPO 比率

对状态动作 `(s_t, a_t)`：

```text
r_t(theta) = pi_theta(a_t | s_t) / pi_old(a_t | s_t)
```

`r_t > 1` 表示新策略提高了该动作概率，`r_t < 1` 表示降低。

### 2. Clipped Policy Loss

要最大化：

```text
L_clip =
E[min(
    r_t(theta) * A_t,
    clip(r_t(theta), 1-epsilon, 1+epsilon) * A_t
)]
```

实现为最小化时取负号：

```text
policy_loss = -L_clip
```

当优势 `A_t > 0` 时，希望提高动作概率，但最多奖励到 `1+epsilon`；当 `A_t < 0` 时，希望降低概率，但限制更新不要过猛。`min` 与 clip 共同形成保守下界。

完整 PPO 常写为：

```text
loss =
policy_loss
+ c_v * value_loss
- c_e * entropy
+ beta * KL
```

- `value_loss` 拟合 return；
- entropy 鼓励探索；
- LLM RL 中常额外对 reference policy 加 KL 约束。

### 3. GAE

先计算 TD residual：

```text
delta_t = r_t + gamma * V(s_(t+1)) - V(s_t)
```

GAE 优势为：

```text
A_t^GAE =
delta_t
+ gamma*lambda*delta_(t+1)
+ (gamma*lambda)^2*delta_(t+2)
+ ...
```

递推实现：

```text
A_t = delta_t + gamma * lambda * A_(t+1)
```

从序列末尾向前计算，并在终止状态或 padding 处用 mask 截断。Return 可由：

```text
R_t = A_t + V(s_t)
```

得到。

### 4. gamma 与 lambda

- `gamma` 控制未来奖励折扣；
- `lambda=0` 接近一步 TD，偏差高、方差低；
- `lambda=1` 接近 Monte Carlo，偏差低、方差高；
- 中间值在两者间折中。

LLM 中一个 response 的 token 可视为多个时间步，最终 reward 常在序列末端给出，同时将 KL 惩罚分布到 token。必须正确处理 EOS、response mask 和不同长度。

### 5. PyTorch 风格伪代码

```python
advantages = torch.zeros_like(rewards)
gae = 0.0

for t in reversed(range(T)):
    next_value = values[:, t + 1]
    non_terminal = 1.0 - dones[:, t]
    delta = (
        rewards[:, t]
        + gamma * next_value * non_terminal
        - values[:, t]
    )
    gae = delta + gamma * lam * non_terminal * gae
    advantages[:, t] = gae * response_mask[:, t]

returns = advantages + values[:, :T]
```

这里假设 `values` 比奖励多一个 bootstrap 位置，`dones` 标记环境终止，`response_mask` 仅屏蔽 prompt 和 padding；三者不能混为一谈。

### 6. 跨境支付场景

若训练支付调查 Agent，奖励可来自工具正确性、事实一致性和合规 verifier。越权资金操作应由环境直接禁止，不能只给负 reward；真实支付链路也不适合作为在线探索环境。

### 7. 面试总结

PPO clip 约束 `pi_theta/pi_old` 的更新幅度，GAE 对 TD residual 做 `(gamma*lambda)` 加权。面试时应能写出两个公式，并解释正负优势、终止 mask、value loss 和 KL 的作用。
