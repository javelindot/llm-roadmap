# LLMs 归一化位置

## 答案

**一句话**：归一化"位置"主要有 **Post-LN / Pre-LN / Sandwich-LN / DeepNorm** 四种，**现代 LLM 默认 Pre-LN + RMSNorm**，特殊场景用 DeepNorm 撑超深网络。

### 1. 四种归一化位置

**Post-LN（原始 Transformer 2017）**
```
y = LN(x + Sublayer(x))
```
- Norm 在残差**之后**
- 浅层稳，深层易爆，需要 warmup
- BERT 时代主流

**Pre-LN（GPT-2 起 / 现代 LLM 主流）**
```
y = x + Sublayer(LN(x))
```
- Norm 在子层**之前**
- 深层稳，warmup 可省
- GPT、Llama、Qwen、Mistral 都用

**Sandwich-LN**
```
y = LN_out(x + Sublayer(LN_in(x)))
```
- 同时在子层前后加 norm
- CogVLM、GLM 用过
- 稳定性 + 表达力兼顾

**DeepNorm（Microsoft）**
```
y = LN(α·x + Sublayer(x))
```
- Post-LN 风格但残差路径加缩放系数 α
- 可稳定训 1000 层 Transformer
- 主要给科研用

### 2. 归一化"算什么" → 两种主流
| 类型 | 公式 | 区别 |
|------|------|------|
| **LayerNorm** | (x - μ) / σ × γ + β | 减均值再缩放 |
| **RMSNorm** | x / RMS(x) × γ | 不减均值，只 RMS |

RMSNorm 比 LN **快 10-20%、精度几乎不掉**，是现代 LLM 的默认选择。

### 3. 主流 LLM 的选择
| 模型 | 归一化位置 | 类型 |
|------|----------|------|
| BERT | Post-LN | LayerNorm |
| GPT-2/3 | Pre-LN | LayerNorm |
| Llama-2/3 | Pre-LN | **RMSNorm** |
| Qwen/Qwen2/Qwen3 | Pre-LN | **RMSNorm** |
| Mistral | Pre-LN | **RMSNorm** |
| GLM | Sandwich-LN | LayerNorm |

### 4. 为什么 Pre-LN 主导
- 训练更稳：残差恒等流，梯度好传
- 无需精心 warmup
- 调参余地大
- 代价：最终精度略低，但工程友好性碾压

### 5. 实战经验
- 默认 Pre-LN + RMSNorm
- 如果训超深（>100 层）→ DeepNorm
- 如果做多模态、追稳定 → 可以 Sandwich-LN
- 自己加分支头 → 也用 Pre-LN 风格更稳

### 6. 一句话总结
**Pre-LN + RMSNorm = 现代 LLM 的归一化默认套件**——稳、快、好调；Post-LN 是历史遗留，DeepNorm/Sandwich 是科研变体。

## 结合项目谈（口语版 · 约 40 秒）

> 我们跨境支付训练用的 base（Qwen、Llama 系列）都是 **Pre-LN + RMSNorm**，所以平时不会专门去动归一化结构。但有一个具体的细节我们踩过：
>
> 我们给客服模型加过一个"满意度预测辅助头"——从最后一层 hidden state 出一个回归头预测客户满意度。第一版我加了**额外的 Post-LN**（出 hidden → LayerNorm → MLP），结果训了一夜 loss 几乎不动；改成 **Pre-LN 风格**（hidden → 用模型原本的 RMSNorm 再走 MLP），loss 立刻收敛，最终 MSE 比 Post-LN 版本好不少。
>
> 这个小实验让我对 "**为什么现代 LLM 默认 Pre-LN**" 有了实感——不是论文挑出来的最优解，而是工程实践筛出来的稳定解，反复证明深层模型里 Pre-LN 梯度好传。
>
> 还有一点：我们部分推理服务把 LayerNorm 全部换成 RMSNorm（其实 Qwen 默认就是），单步推理快了 8% 左右，质量基本一样。**这是免费的性能**，没理由不开。
>
> 一句话总结：**归一化位置是个"能让模型训得起来"的基础设施问题**，选错了梯度直接死，工程上无脑 Pre-LN + RMSNorm 就对了。

**🪝 顺嘴带出去的钩子**：
- "RMSNorm 在推理优化里能省多少算力可以聊。"
- "辅助头在大模型上怎么接稳的几个经验。"
- "DeepNorm 在金融领域有没有用武之地我们讨论过。"
