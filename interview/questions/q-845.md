# 问RLHF的流程，问到RM的训练的时候突然问RLHF的loss是什么，顺势就以为问的是RM的loss，答交叉熵，说不对？？？不是吗？后面想了一下他可能是想问PPO的loss

## 答案

“RLHF 的 loss”不是一个唯一损失，应先反问面试官指 SFT、Reward Model 还是 PPO 阶段。回答 RM 用交叉熵并没有原则性错误，但要具体说是 pairwise logistic cross entropy。

### 1. SFT Loss

\[
\mathcal L_{\rm SFT}
=-\sum_t\log\pi_\theta(y_t|x,y_{<t})
\]

即 assistant token 的 next-token cross entropy。

### 2. Reward Model Loss

对 chosen \(y_w\) 和 rejected \(y_l\)：

\[
\mathcal L_{\rm RM}
=-\log\sigma(
r_\phi(x,y_w)-r_\phi(x,y_l))
\]

这就是 Bradley-Terry 偏好模型的二元交叉熵/负对数似然。若有多候选，也可用 listwise ranking loss。

### 3. PPO Policy Loss

令：

\[
r_t(\theta)
=\frac{\pi_\theta(a_t|s_t)}
{\pi_{\rm old}(a_t|s_t)}
\]

则 clipped objective：

\[
\mathcal L_{\rm policy}
=-\mathbb E\left[
\min(r_tA_t,
\operatorname{clip}(r_t,1-\epsilon,1+\epsilon)A_t)
\right]
\]

训练还通常包含：

- value/critic loss：\((V_\psi-G_t)^2\)；
- entropy bonus，鼓励探索；
- 相对 reference policy 的 KL penalty；
- 总 reward 由 RM、规则和 KL 等组合。

因此实际总目标类似：

\[
\mathcal L
=\mathcal L_{\rm policy}
+c_v\mathcal L_{\rm value}
-c_eH
\]

KL 可能作为 reward shaping 或独立项实现。

### 面试总结

RLHF 各阶段 loss 不同：SFT 是 token CE，RM 是 pairwise logistic CE，PPO 是 clipped policy loss 加 value、entropy 和 KL。遇到含糊提问先确认阶段，再给公式。
