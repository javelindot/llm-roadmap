# 用两个栈实现队列

## 答案

使用输入栈 `in` 和输出栈 `out`：

- 入队时压入 `in`；
- 出队或查看队首时，若 `out` 为空，将 `in` 全部弹出并压入 `out`；
- 若 `out` 非空，直接使用其栈顶；
- 只有 `out` 为空时才搬运，不能每次出队都来回倒。

```java
import java.util.ArrayDeque;
import java.util.Deque;
import java.util.NoSuchElementException;

final class TwoStackQueue<E> {
    private final Deque<E> in = new ArrayDeque<>();
    private final Deque<E> out = new ArrayDeque<>();

    public void offer(E value) {
        in.push(value);
    }

    public E poll() {
        moveIfNeeded();
        if (out.isEmpty()) {
            throw new NoSuchElementException("queue is empty");
        }
        return out.pop();
    }

    public E peek() {
        moveIfNeeded();
        if (out.isEmpty()) {
            throw new NoSuchElementException("queue is empty");
        }
        return out.peek();
    }

    public boolean isEmpty() {
        return in.isEmpty() && out.isEmpty();
    }

    private void moveIfNeeded() {
        if (out.isEmpty()) {
            while (!in.isEmpty()) {
                out.push(in.pop());
            }
        }
    }
}
```

每个元素最多进入、离开两个栈各一次，因此 `offer` 为 \(O(1)\)，`poll/peek` 摊还 \(O(1)\)，最坏单次搬运 \(O(n)\)，空间 \(O(n)\)。该实现不是线程安全的，并发场景需外部同步或使用并发队列。

### 面试总结

双栈队列利用第二个栈反转顺序，并采用惰性搬运。复杂度重点是出队最坏 \(O(n)\)、摊还 \(O(1)\)，而不是声称每次都严格 \(O(1)\)。
