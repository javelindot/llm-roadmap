# ResNet如何解决梯度消失问题？残差连接的作用是什么？

## 答案

**一句话**：ResNet 通过残差连接让网络学习残差函数，并为梯度提供更短路径，从而缓解深层网络的梯度消失和退化问题。

### 核心回答
普通深层网络要直接学习 `H(x)`，ResNet 改成学习：

```text
H(x) = F(x) + x
```

其中 `x` 是跳跃连接，`F(x)` 是残差。

作用：
- 梯度可以沿 shortcut 直接回传。
- 深层网络更容易优化。
- 如果某层没用，可以学成接近 0，让整体接近恒等映射。
- 解决网络加深后训练误差反而变差的退化问题。

### 1. 梯度消失和退化问题的区别

梯度消失指深层网络反向传播时，前面层梯度越来越小，导致学不动。退化问题是网络变深后，理论表达能力更强，但训练误差反而变高，说明优化更困难。

ResNet 用 shortcut 给信息和梯度提供直通路径，让网络不用每一层都强行学习复杂变换。

### 2. 残差为什么容易优化

与其直接学习 `H(x)`，不如学习残差 `F(x)=H(x)-x`。如果某些层暂时没学到有用变换，可以让 `F(x)` 接近 0，整体仍接近恒等映射。

### 3. 面试总结

残差连接的核心价值是让深层网络更容易训练，不只是解决梯度消失，也缓解网络加深后的优化退化。

### 4. 结合跨境支付项目怎么说（面试口径）

> 如果项目里做证照图片识别或商户材料分类，ResNet 的残差思想很实用。证照图像里既有低层文字边缘，也有高层版式语义，网络加深能提更多特征，但太深又难训。
>
> 残差连接让深层视觉模型更稳定。这个思想其实也影响了 Transformer，大模型里每个 attention 和 FFN 外面也有残差连接，否则很难堆到几十层甚至上百层。
>
> 所以我会把 ResNet 当成深层模型训练稳定性的经典例子，而不仅仅是一个 CNN 架构。

**钩子**：残差连接不仅用于 CNN，也是现代深层模型稳定训练的基础设计。
