# boosting和bagging在特征方面的区别，(没做过，...

## 答案

Bagging 和 Boosting 的核心区别不在“是否使用特征”，而在 **样本、特征和弱学习器如何组织**。

### 1. Bagging

- 对训练样本做 Bootstrap 抽样。
- 多个基学习器相互独立，可并行训练。
- 最后做投票或平均，主要降低方差。

Random Forest 还会在每个节点只考虑随机特征子集，因此同时引入：

- 样本随机性；
- 特征随机性。

这样可以降低树之间的相关性。

### 2. Boosting

- 基学习器按顺序训练。
- 后一个学习器重点修正前面模型的错误或残差。
- 最后加权组合，主要降低偏差，也能降低部分方差。

AdaBoost 通过提高错分样本权重聚焦难例；GBDT/XGBoost/LightGBM 拟合损失函数的负梯度或残差。

### 3. 在特征方面的区别

| 方面 | Bagging / Random Forest | Boosting |
|------|-------------------------|----------|
| 特征使用 | RF 常对特征随机采样 | 通常由每棵树按增益选择 |
| 学习关系 | 各模型独立 | 后续模型依赖前面残差 |
| 列采样 | RF 是核心随机性之一 | XGBoost 等也可配置列采样 |
| 特征重要性 | 多棵独立树统计平均 | 累计各轮分裂贡献 |

所以不能简单说“Bagging 用部分特征、Boosting 用全部特征”。现代 Boosting 也常做 Column Subsampling。

### 4. 选型

- 数据噪声大、模型容易过拟合、希望并行：Bagging。
- 需要更高精度、愿意顺序训练和细调参数：Boosting。
- Boosting 对异常值和错误标签可能更敏感，因为它会持续关注难例。

**面试总结**：Bagging 通过独立和随机化降低相关性，Boosting 通过逐轮纠错提升能力；特征采样只是实现手段之一。
