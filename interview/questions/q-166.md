# 扩散模型在训练模型时，预测目标是什么？

## 答案

**一句话**：扩散模型训练时的预测目标可以是噪声 epsilon、原始图像 x0，或者 v-parameterization，不同目标对应不同稳定性和采样表现。

### 核心回答
常见预测目标：
- **预测噪声 epsilon**：最经典，模型预测加入了什么噪声。
- **预测 x0**：直接预测干净样本。
- **预测 v**：结合 x0 和噪声的一种参数化，很多现代扩散模型使用，稳定性好。

训练时：
1. 从真实数据取 `x0`。
2. 采样时间步 `t` 和噪声。
3. 得到带噪样本 `x_t`。
4. 模型根据 `x_t` 和 `t` 预测目标。
5. 用 MSE 训练。

### 1. 不同预测目标的直观区别

扩散模型训练的关键是让模型知道"当前带噪样本应该往哪个方向去噪"。预测噪声是最经典做法，因为前向过程加了什么噪声是已知的。

预测 `x0` 则更直接，模型输出干净样本；预测 `v` 是一种折中参数化，在一些模型里训练和采样更稳定。

### 2. 为什么要带时间步

不同时间步的噪声强度不同。早期时间步图像还比较清楚，后期接近纯噪声。模型必须知道当前处在哪个时间步，才能选择合适的去噪策略。

### 3. 面试总结

扩散模型训练时可以预测噪声、x0 或 v，本质都是学习逆向去噪方向。

### 4. 结合跨境支付项目怎么说（面试口径）

> 对业务来说，我更关心它能不能可靠地去噪或生成。比如证照扫描件增强，如果模型预测的是噪声，本质是在学怎么把噪声从图片里去掉。
>
> 但支付材料审核不能让模型"脑补"不存在的文字，所以这种能力更适合做图像质量增强或辅助预处理，最终字段识别和真伪判断还要靠 OCR、规则和人工复核。
>
> 我的原则是：图像增强可以帮助人和 OCR 看得更清楚，但不能改变业务事实。

**钩子**：生成和去噪能力要服务业务预处理，不能替代证照真实性判断。
