# 一面问了论文和实习项目 题目是手写cross attention

## 答案

Cross-Attention 中，Query 来自目标序列，Key/Value 来自另一个源序列。它用于 Encoder-Decoder、文本-图像融合、Query-Document 交互等。

### 1. 数学形式

```text
Q = X_query W_Q
K = X_context W_K
V = X_context W_V

Attention =
softmax(QK^T / sqrt(d_head) + mask) V
```

若：

```text
X_query:   [B, Lq, D]
X_context: [B, Lk, D]
```

输出是：

```text
[B, Lq, D]
```

### 2. PyTorch 手写

```python
import math
import torch
import torch.nn as nn


class CrossAttention(nn.Module):
    def __init__(self, hidden_size, num_heads, dropout=0.0):
        super().__init__()
        assert hidden_size % num_heads == 0

        self.num_heads = num_heads
        self.head_dim = hidden_size // num_heads

        self.q_proj = nn.Linear(hidden_size, hidden_size)
        self.k_proj = nn.Linear(hidden_size, hidden_size)
        self.v_proj = nn.Linear(hidden_size, hidden_size)
        self.o_proj = nn.Linear(hidden_size, hidden_size)
        self.dropout = nn.Dropout(dropout)

    def _split_heads(self, x):
        batch, length, hidden = x.shape
        x = x.view(
            batch, length, self.num_heads, self.head_dim
        )
        return x.transpose(1, 2)

    def forward(self, query, context, context_mask=None):
        q = self._split_heads(self.q_proj(query))
        k = self._split_heads(self.k_proj(context))
        v = self._split_heads(self.v_proj(context))

        scores = torch.matmul(
            q, k.transpose(-2, -1)
        ) / math.sqrt(self.head_dim)

        if context_mask is not None:
            # context_mask: [B, Lk], 1 表示有效
            mask = context_mask[:, None, None, :].bool()
            scores = scores.masked_fill(
                ~mask, torch.finfo(scores.dtype).min
            )

        weights = torch.softmax(scores, dim=-1)
        weights = self.dropout(weights)

        output = torch.matmul(weights, v)
        output = output.transpose(1, 2).contiguous()

        batch, length, _, _ = output.shape
        output = output.view(batch, length, -1)
        return self.o_proj(output), weights
```

### 3. 与 Self-Attention 区别

- Self-Attention：Q/K/V 来自同一序列；
- Cross-Attention：Q 与 K/V 来自不同序列；
- Decoder Cross-Attention 通常不需要 causal mask，但需要 context padding mask；
- 输出长度等于 Query 长度。

### 4. 易错点

- `hidden_size` 必须能整除 head 数；
- mask 广播形状；
- softmax 维度是 key length；
- `transpose` 后需要 `contiguous` 再 view；
- 混合精度下 mask 使用有限最小值；
- context 全被 mask 时需要额外处理，避免 NaN。

### 5. 跨境支付场景

可让工单 query token 关注政策/订单证据 token，或让文本 token 关注凭证图像特征。但权限过滤和实时状态必须在送入 context 前完成。

### 6. 面试总结

Cross-Attention 的关键是 Query 和 Context 来源不同，输出跟随 Query 长度。手写时重点展示 shape、缩放、mask、多头拆分与合并。
