# 视频大模型的长时序建模方案

## 答案

视频大模型的长时序难点是帧数巨大、动作跨时间尺度、信息冗余高，无法把所有高分辨率帧 token 直接送入标准全局注意力。

### 时空压缩

先用视觉 encoder 提取帧或短 clip 特征，再进行空间和时间降采样。常见方法包括 tubelet embedding、temporal pooling、token merging、learned query/Q-Former 和关键帧选择。静态片段可低采样，动作密集片段提高采样率。

### 分层建模

局部层在短 clip 内学习运动和细节，长程层对 clip summary 建模。可使用 factorized attention，先做空间注意力再做时间注意力，将完整时空二次复杂度拆开；也可用 sliding-window、稀疏全局 token 和分层 Transformer。

对小时级视频，可构建“帧或镜头、事件、章节”多级记忆，查询时先定位相关章节，再读取细粒度片段，而不是全量输入。

### 外部记忆与检索

将视频切成带时间戳的片段，离线生成视觉、ASR、OCR 和音频 embedding，并建立向量与关键词索引。问题到来时先检索候选片段，再让多模态模型推理。在线 Agent 可迭代执行时间定位、片段读取和证据核验。

### 训练与评估

训练数据应覆盖动作顺序、跨片段因果、时间定位和长程问答。可使用短到长 curriculum、随机帧丢弃和跨片段对比学习。位置编码需要支持长度扩展，并保留绝对时间戳。

评估除问答准确率外，还要看 temporal grounding、事件顺序、needle-in-video、不同视频长度的性能和计算成本。均匀采更多帧不是充分方案，关键是内容自适应压缩、层级记忆和按需检索。
