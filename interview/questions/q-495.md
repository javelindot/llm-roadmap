# 生成模型了解哪些，GLM结构介绍一下

## 答案

### 1. 常见生成模型

- **自回归模型**：按顺序预测下一个 Token，如 GPT。
- **VAE**：学习概率潜空间，通过编码—采样—解码生成。
- **GAN**：生成器与判别器对抗训练，适合高质量图像生成。
- **Diffusion**：学习逐步去噪，当前图像/视频生成主流之一。
- **Normalizing Flow**：使用可逆变换，能精确计算似然。
- **自编码式 / Masked 模型**：通过恢复被遮盖内容学习表示或生成。

### 2. GLM 的核心思想

原始 GLM 是一种基于 Transformer 的 **Autoregressive Blank Infilling** 预训练框架，试图统一自然语言理解和生成。

训练时先在原文本中遮盖一个或多个连续 Span，再让模型自回归地生成被遮盖内容：

```text
原文：A [MASK] B [MASK] C
目标：依次生成每个空白中的 Token
```

### 3. 注意力与位置设计

GLM 使用单个 Transformer，通过 Attention Mask 同时实现：

- 对可见上下文的双向理解；
- 对被遮盖 Span 内 Token 的自回归生成。

它还使用二维位置编码，分别表达：

- 被遮盖 Span 在原序列中的位置；
- Token 在该 Span 内部的生成顺序。

多个空白的生成顺序可随机打乱，减少模型依赖固定顺序。

### 4. 与 BERT、GPT 的关系

- BERT 使用双向 Encoder 和 MLM，擅长理解，但不天然适合连续生成。
- GPT 使用因果 Decoder 和 Next Token Prediction，擅长生成。
- GLM 用 Blank Infilling 把双向上下文理解与自回归生成结合起来。

### 5. 回答时的注意点

“GLM”可能指原始 GLM 预训练框架，也可能泛指后续 ChatGLM 系列。不同代际的 ChatGLM 架构细节会变化，面试中应先确认对方问的是论文原理还是具体模型版本。

**面试总结**：GLM 的关键不是简单的 Mask，而是“用双向上下文定位空白，再在空白内部自回归生成”，从而统一理解和生成任务。
