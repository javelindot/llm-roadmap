# Normalization有哪些，有什么作用

## 答案

Normalization 的主要作用是稳定激活尺度、改善梯度传播、允许更大的学习率，并降低网络对初始化和数据分布变化的敏感度。

### 1. 常见归一化

| 方法 | 统计维度 | 常见场景 |
|------|----------|----------|
| BatchNorm | Batch 内同一通道 | CNN |
| LayerNorm | 单样本特征维 | Transformer、RNN |
| RMSNorm | 单样本均方根 | 现代 LLM |
| InstanceNorm | 单样本单通道空间维 | 风格迁移 |
| GroupNorm | 通道分组 | 小 Batch 视觉任务 |
| SyncBatchNorm | 多卡同步 Batch 统计 | 分布式视觉训练 |

### 2. BatchNorm

训练时使用当前 Batch 均值方差，并维护移动统计量供推理使用。它有一定正则化效果，但依赖 Batch Size 和数据分布。

### 3. LayerNorm 与 RMSNorm

LayerNorm 对单个 Token/样本的隐藏维归一化，不依赖 Batch。RMSNorm 不减均值，只按均方根缩放，计算更简单。

### 4. Transformer 中的位置

- Post-LN：`Norm(x + Sublayer(x))`
- Pre-LN：`x + Sublayer(Norm(x))`

Pre-LN 通常更利于深层模型梯度传播；最终性能还与初始化和残差设计有关。

### 5. Normalization 不能解决什么

- 不能替代正确的数据预处理和学习率设置。
- 不保证消除梯度爆炸，仍可能需要 Gradient Clipping。
- BatchNorm 的训练/推理统计不一致可能带来分布偏移。
- 归一化维度选错会破坏有意义的尺度信息。

### 6. Norm 与标准化输入的区别

输入标准化是对数据集特征预处理；网络中的 Normalization 是对中间激活动态调整，两者作用层级不同。

**面试总结**：选择哪种 Norm 取决于数据形状和 Batch 条件，Transformer 通常使用与 Batch 无关的 LayerNorm/RMSNorm。
