# 看一下你的 AI 基础。介绍一下 transformer 基本原理？解释一下注意力机制？解释一下什么是扩散模型？

## 答案

这题包含 Transformer、注意力和扩散模型三部分。

### 1. Transformer 基本原理

Transformer 用 Attention 代替 RNN 的顺序递归。原始结构分 Encoder 和 Decoder：

- Encoder：双向 Self-Attention + FFN；
- Decoder：Causal Self-Attention + Cross-Attention + FFN；
- 每个子层配 Residual 和 LayerNorm；
- 使用位置编码表示顺序。

现代 LLM 多为 Decoder-only，通过 next-token prediction 训练。

### 2. 注意力机制

输入投影为：

```text
Q = XW_Q
K = XW_K
V = XW_V
```

Scaled Dot-Product Attention：

```text
Attention(Q,K,V) =
softmax(QK^T / sqrt(d_k)) V
```

- Q 表示当前位置想查询什么；
- K 表示每个位置可匹配的特征；
- V 表示实际聚合的信息；
- 除以 `sqrt(d_k)` 防止点积过大导致 softmax 饱和；
- mask 用于屏蔽 padding 或未来 token。

Multi-Head Attention 在多个子空间并行学习不同关系。

### 3. Transformer 优缺点

优点：

- 训练并行；
- 长距离依赖；
- 易扩展；
- 可统一文本、视觉、音频。

缺点：

- 标准 Attention 对长度近似二次复杂度；
- KV Cache 占用高；
- 需要大量数据和计算；
- 输出不保证事实正确。

### 4. 扩散模型是什么

扩散模型通过“加噪”和“去噪”学习数据分布。

**Forward Process**

逐步向真实样本 `x_0` 加高斯噪声：

```text
x_0 -> x_1 -> ... -> x_T ≈ Gaussian noise
```

**Reverse Process**

训练网络预测噪声、score、velocity 或干净样本，逐步从噪声还原数据：

```text
x_T -> ... -> x_1 -> x_0
```

经典噪声预测目标：

```text
L = E[||epsilon - epsilon_theta(x_t, t, condition)||^2]
```

### 5. 条件生成

文本条件常由 Text Encoder 编码，再通过 Cross-Attention 注入 U-Net 或 DiT。Classifier-Free Guidance 组合有条件与无条件预测，加强提示遵循。

现代模型大量使用 Diffusion Transformer/DiT，在 latent space 中扩散，降低像素级计算。

### 6. Transformer 与扩散的关系

Transformer 是网络架构/序列建模机制；Diffusion 是生成建模和训练/采样范式。扩散模型内部可以用 U-Net，也可以用 Transformer。

### 7. 跨境支付场景

Transformer 适合文档、客服和工具 Agent；扩散模型可生成营销素材或合成测试图像，但证件、支付凭证等不能用生成内容冒充真实材料，需水印和审核。

### 8. 面试总结

Transformer 通过 Self-Attention 建模 token 关系，注意力是 Q/K 匹配后对 V 加权；扩散模型通过学习逆向去噪从随机噪声生成样本。两者属于不同层次的概念，可以组合使用。
