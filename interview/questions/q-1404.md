# 6.激活函数和损失函数区别

## 答案

激活函数和损失函数作用不同：

- **激活函数**位于网络层内部或输出层，将线性变换变成非线性表示；
- **损失函数**位于模型输出与标签之间，衡量预测误差并提供训练优化目标。

### 1. 激活函数

神经网络一层通常写成：

```text
z = Wx + b
h = activation(z)
```

如果没有非线性激活，多层线性变换仍等价于一层线性变换，模型无法学习复杂决策边界。

常见激活函数：

- Sigmoid：输出 0 到 1；
- Tanh：输出 -1 到 1；
- ReLU：`max(0, x)`；
- Leaky ReLU；
- GELU；
- SiLU/Swish；
- SwiGLU：现代大模型 FFN 常用门控激活。

### 2. 损失函数

损失函数衡量：

```text
prediction 与 target 的差异
```

常见损失：

- 回归：MSE、MAE、Huber；
- 二分类：Binary Cross Entropy；
- 多分类：Cross Entropy；
- 排序：pairwise hinge/BPR、listwise loss；
- 对比学习：InfoNCE；
- 语言模型：next-token cross entropy。

优化器通过损失对参数求梯度：

```text
theta <- theta - learning_rate * grad_theta(L)
```

### 3. 输出激活和 loss 的配合

**二分类**

```text
logit -> sigmoid -> BCE
```

实际框架通常使用数值更稳定的 `BCEWithLogitsLoss`，直接输入 logit，不要手动重复 sigmoid。

**多分类**

```text
logits -> softmax -> Cross Entropy
```

PyTorch `CrossEntropyLoss` 内部已包含 `log_softmax`，不应先手动 softmax。

**回归**

输出层可不加激活，再使用 MSE/MAE；如果目标有范围约束，可按业务使用 sigmoid、softplus 等。

### 4. 二者如何共同影响梯度

损失函数定义“优化什么”，激活函数影响“网络如何表达、梯度如何传播”。例如 Sigmoid 在大绝对值区间容易饱和，隐藏层大量使用时可能梯度变小；ReLU 简单高效，但可能出现负区间长期无梯度的死节点。

### 5. 跨境支付场景

- 欺诈二分类：输出 logit + BCE；
- 风险等级多分类：logits + Cross Entropy；
- 到账时间回归：线性/正值输出 + MAE/Huber；
- 支付通道排序：排序 loss；
- 商户客服大模型：token logits + Cross Entropy。

损失还可以做类别权重或 focal loss，以提高稀有高风险样本的关注度，但要检查概率校准。

### 6. 面试总结

激活函数提供非线性表达并影响梯度传播，损失函数衡量预测与目标的差距并定义优化方向。回答时还要讲清输出层与 loss 的正确配对，避免重复使用 sigmoid 或 softmax。
