# 介绍一下self-attention及其适用场景

## 答案

**一句话**：Self-Attention 是序列内部每个 token 和其它 token 计算相关性并聚合信息的机制，适合文本理解、生成、图像 patch 建模和多模态对齐。

### 1. 原理

Self-Attention 中 Q、K、V 都来自同一输入序列。每个 token 都可以关注序列中其它 token，从而获得上下文表示。

公式：

```text
softmax(QK^T / sqrt(d_k))V
```

### 2. 适用场景

- NLP：文本分类、生成、翻译、问答。
- RAG：query 和文档语义理解。
- 代码：上下文依赖理解。
- 图像：ViT 把图片切成 patch 后做 self-attention。
- 多模态：文本和图像特征建模。

### 3. 结合跨境支付项目怎么说（面试口径）

> 在跨境支付项目里，Self-Attention 可以帮助模型理解一段复杂问题里的关键关系。比如“用户在巴西发起 USD payout，通道返回账户无效”，模型需要同时理解国家、币种、支付方式、错误原因之间的关系。
>
> 它适合处理客服对话、通道文档、风控解释和 KYC 材料文本。但注意力只解决语义理解问题，真实订单状态仍要通过工具查询。
