# 什么时候用batch norm什么时候用layer norm，为什么

## 答案

**一句话**：BatchNorm 适合 CNN 和 batch 稳定的场景，LayerNorm 适合 Transformer、RNN、变长序列和小 batch 场景。

### 区别

- BatchNorm：跨 batch 统计，依赖 batch 大小，训练和推理统计不同。
- LayerNorm：对单样本 hidden 维度归一化，不依赖 batch，训练推理一致。

### 选择

- 图像 CNN：常用 BatchNorm。
- NLP/Transformer：常用 LayerNorm。
- 小 batch/变长序列/自回归生成：LayerNorm 更稳定。

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付模型服务里，LLM 推理请求长短不一，batch 也不稳定，所以 Transformer 使用 LayerNorm 更合适。对于图像材料识别，如果用 CNN 检测证件或回单，BatchNorm 仍然常见。
>
> 选择归一化方式要看模型结构和部署场景。

**钩子**：可以引到“CNN 用 BN、Transformer 用 LN、变长序列和小 batch”。
