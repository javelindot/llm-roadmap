# 介绍一下 ALBEF 、 BLIP

## 答案

ALBEF 和 BLIP 都是 Salesforce 提出的视觉语言预训练方法。ALBEF 重点解决图文 token 未对齐就直接融合的问题；BLIP 在统一理解与生成架构之外，重点解决网页图文数据噪声。

### ALBEF

ALBEF 即 Align Before Fuse。它包含图像 encoder、文本 encoder 和跨模态 encoder：

1. 图像与文本先独立编码。
2. 使用 Image-Text Contrastive loss 将全局图文表示拉到同一空间。
3. 再通过 cross-attention 融合，训练 Image-Text Matching 和 Masked Language Modeling。

它使用 momentum encoder 和队列生成更稳定的对比目标，并通过 momentum distillation 为噪声网页数据提供软伪标签。其核心观点是先做表示对齐，再做细粒度融合。

### BLIP

BLIP 使用 Multimodal Mixture of Encoder-Decoder：

- 单模态 image/text encoder 支持对比学习；
- image-grounded text encoder 通过 cross-attention 支持理解和匹配；
- image-grounded text decoder 使用因果 self-attention 支持图像描述生成。

三种功能共享大部分参数，通过 ITC、ITM 和 Language Modeling 联合预训练，实现理解与生成统一。

BLIP 还提出 CapFilt：captioner 为图像生成合成描述，filter 过滤原始和合成图文对中的噪声，通过 caption bootstrapping 改善训练数据质量。

### 对比

ALBEF 更强调对齐后融合与 momentum distillation；BLIP 继承图文对齐/匹配思想，同时引入可生成 decoder 和数据自举，因此能统一检索、VQA 与 captioning。还应区分 BLIP 与 BLIP-2，后者使用冻结图像 encoder、冻结 LLM 和 Q-Former 连接两者。
