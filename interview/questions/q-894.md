# 模型评估性能指标有哪些以及它们的定义和用途

## 答案

指标必须按任务、类别分布和业务成本选择，不能只报 Accuracy。

### 1. 分类

设 TP、FP、TN、FN：

\[
Accuracy=\frac{TP+TN}{N}
\]

\[
Precision=\frac{TP}{TP+FP},\quad
Recall=\frac{TP}{TP+FN}
\]

\[
F1=\frac{2PR}{P+R}
\]

- Accuracy：类别均衡、误判成本接近；
- Precision：误报代价高；
- Recall：漏报代价高；
- F1：平衡 Precision/Recall；
- Macro-F1：每类等权，关注长尾；
- Micro-F1：全样本汇总，受大类影响。

ROC-AUC 衡量随机正样本得分高于随机负样本的概率；类别极不均衡时 PR-AUC 更能反映正类效果。

### 2. 回归

- MAE：平均绝对误差，对异常值较鲁棒；
- MSE/RMSE：更重罚大误差；
- \(R^2\)：相对均值基线解释方差；
- MAPE：直观百分比，但真实值接近零时不稳定。

### 3. 排序与检索

- Recall@K/HitRate@K：相关项是否进入候选；
- Precision@K：前 K 中相关项比例；
- MRR：第一个相关结果排名倒数；
- MAP：多个相关项的平均精度；
- NDCG@K：考虑相关性等级与位置折损；
- Coverage、diversity、novelty：推荐长尾与多样性。

### 4. 生成、RAG 与 Agent

- BLEU/ROUGE/BERTScore：有参考文本时的相似性；
- correctness、faithfulness、citation accuracy；
- 工具选择/参数正确率、任务完成率、违规率；
- 人工或校准后的 LLM Judge；
- 延迟、token 成本和稳定性。

### 5. 校准与系统

- LogLoss/NLL、Brier Score、ECE：概率是否可信；
- P50/P95/P99、QPS、错误率、显存和单位成本；
- 线上 A/B：转化、解决率、投诉和长期留存。

跨境支付风控需同时看 PR-AUC、风险召回、正常交易误杀、校准和人工审核量。

### 面试总结

指标选择由任务与错误成本决定：分类看 P/R/F1/AUC，回归看 MAE/RMSE，检索看 Recall/MRR/NDCG，生成看正确性和忠实度，线上还必须看 P99、成本与业务结果。
