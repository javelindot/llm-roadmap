# 22、算法：力扣240 搜索二维矩阵 II

## 答案

LeetCode 240 中，每行从左到右递增，每列从上到下递增。可以从右上角或左下角开始，每次排除一行或一列。

### 1. 右上角思路

当前位置为 `matrix[row][col]`：

- 等于 target：找到；
- 大于 target：当前列下面都更大或不小，向左；
- 小于 target：当前行左边都更小或不大，向下。

### 2. Python 实现

```python
def search_matrix(matrix, target):
    if not matrix or not matrix[0]:
        return False

    rows = len(matrix)
    cols = len(matrix[0])

    row = 0
    col = cols - 1

    while row < rows and col >= 0:
        value = matrix[row][col]

        if value == target:
            return True
        if value > target:
            col -= 1
        else:
            row += 1

    return False
```

### 3. 复杂度

```text
时间复杂度：O(m+n)
空间复杂度：O(1)
```

每一步至少排除一行或一列，最多移动 `m+n` 次。

### 4. 为什么不能把它当成 LeetCode 74

LeetCode 74 还要求后一行首元素大于前一行尾元素，可把整个矩阵视为一维有序数组做 O(log(mn)) 二分。

LeetCode 240 只保证行列分别有序，不能直接展平成全局有序数组。

### 5. 边界

- 空矩阵；
- 只有一行或一列；
- target 小于最小值或大于最大值；
- 重复元素；
- 非方阵。

### 6. 面试总结

从右上角看，它同时是当前行较大端和当前列较小端，因此比较一次就能安全排除一行或一列，得到 O(m+n)。
