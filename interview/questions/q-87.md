# Xavier 初始化和 He 初始化的适用场景是什么？

## 答案

**一句话**：**Xavier 适合 tanh / sigmoid 这种"对称饱和"激活；He 适合 ReLU 这种"半轴激活"——差别在 He 把 ReLU 砍掉一半的事实考虑进了方差**。

### 1. Xavier / Glorot 初始化（2010）
**目标**：保持前向激活方差和反向梯度方差稳定。

**公式**：
$$W \sim N\left(0, \frac{2}{n_{in} + n_{out}}\right) \quad \text{或} \quad W \sim U\left(-\sqrt{\frac{6}{n_{in}+n_{out}}}, \sqrt{\frac{6}{n_{in}+n_{out}}}\right)$$

**假设**：激活函数关于 0 对称且接近线性（如 tanh 在 0 附近）。

### 2. He / Kaiming 初始化（2015）
**目标**：解决 ReLU 砍掉一半激活带来的方差减半问题。

**公式**：
$$W \sim N\left(0, \frac{2}{n_{in}}\right) \quad \text{或} \quad W \sim U\left(-\sqrt{\frac{6}{n_{in}}}, \sqrt{\frac{6}{n_{in}}}\right)$$

**核心改进**：方差比 Xavier 大一倍，正好补偿 ReLU 让一半神经元死掉的影响。

### 3. 对比表
| 维度 | Xavier | He |
|------|--------|-----|
| 方差 | 2/(n_in + n_out) | 2/n_in |
| 适用激活 | tanh / sigmoid | ReLU / LeakyReLU / GELU |
| 数学假设 | 激活线性区 + 对称 | 半轴激活，输入 0 均值 |
| 用于 LLM | × | 部分（FFN） |

### 4. 为什么 ReLU 需要更大方差
- ReLU(x) = max(0, x)
- 一半负数被砍 → 输出方差 ≈ 输入方差 / 2
- 多层叠加 → 方差指数衰减 → 激活消失
- He 用 2/n_in（Xavier 的两倍）补偿

### 5. LLM 实际使用情况
- **Embedding / Linear**：通常 N(0, 0.02) 截断正态（GPT/Bert 经验值）
- **FFN（SwiGLU/ReLU）**：可以用 He，但很多直接用 0.02
- **Attention Q/K/V**：常用 Xavier 或 0.02
- **LayerNorm γ=1, β=0**：固定，不参与随机初始化

### 6. PyTorch 默认
- `nn.Linear` 默认 Kaiming uniform（He 风格）
- `nn.Conv2d` 默认 Kaiming uniform
- `nn.Embedding` 默认 N(0, 1)（**注意需要重写**）

### 7. 实战经验
- **Transformer 实际不严格按 He/Xavier**，多用 N(0, 0.02) + 输出层缩放 1/√(2L)
- 输出/分类头有时初始化为 0（保持初始输出为均匀分布）
- LoRA 标准：A Kaiming uniform、B 全 0

### 8. 一句话总结
**Xavier 给 tanh / sigmoid 用、He 给 ReLU 用**——LLM 实际多用 N(0, 0.02) 经验值 + 输出层缩放，理论方案不严格使用。

## 结合项目谈（口语版 · 约 25 秒）

> 老实说，工业里**很少自己设 Xavier 或 He**——预训练 base 帮我们做了，微调阶段 LoRA 也是论文标准配方（A Kaiming、B 全 0）。
>
> 但理解原理在两个场景有用过：
>
> **场景一：加分类头**。给 Qwen-7B 加一个意图分类头，第一版用 PyTorch 默认（Kaiming uniform），训得起来但 loss 起点比预期高。改成 N(0, 0.02) 后 loss 起点降一截，收敛也快。原因是 hidden_dim=4096 很大，Kaiming 默认的方差太大让 logits 初始就饱和了。
>
> **场景二：调试梯度问题**。一个实习生写自定义 transformer block 时漏了初始化，结果训了一天 loss 不动。我让他在 forward 前 print 各层激活的方差——发现方差逐层指数衰减，到第 8 层激活几乎全 0。这就是典型的"初始化方差不对"。改成 He 之后立刻收敛。
>
> 我对这道题的总结：**面试要会讲 Xavier 和 He 的差异和原理**，但工业里别背公式，重点是**会 debug "激活/梯度方差异常"这个症状**，比记公式重要。
