# 怎么挖掘这些视频的标签？

## 答案

视频标签挖掘应采用“多模态候选生成、标签体系治理、弱监督训练、人工闭环”，不能只对标题做关键词匹配。

### 1. 建立标签体系

- 定义内容、人物、场景、动作、商品、情绪、安全等标签层级；
- 处理同义词、上下位关系、互斥标签和未知类；
- 明确标签用途：检索、推荐、审核和广告需要的精度不同；
- 为每个标签规定正例、反例与边界案例。

### 2. 多模态信号

- 标题、描述、话题、评论和作者信息；
- ASR 提取语音文本，OCR 提取画面文字；
- 均匀采帧、镜头切分后用视觉模型识别对象和场景；
- 音频事件、音乐与说话人特征；
- 用户搜索、点击、完播、分享等行为信号。

使用多模态 Embedding 或 VLM 生成候选标签，再通过规则、分类器和层级约束融合。对长视频应先按镜头或时间片打局部标签，再汇总视频级标签。

### 3. 数据与训练

- 从人工标签、高置信规则、知识库和教师模型构造弱标签；
- 对长尾标签采用主动学习，优先送人工标注高不确定样本；
- 多标签分类使用 BCE/Focal Loss，层级标签可增加父子一致性约束；
- 通过蒸馏把昂贵 VLM 的能力迁移到线上轻量模型；
- 新热点和新实体用增量词表、聚类与人工审核更新。

### 4. 评估

同时看 micro/macro F1、每标签 precision/recall、覆盖率、层级一致性、时效性和线上检索/推荐收益。高风险审核标签应优先 precision 或 recall，不能只看平均 F1。

### 面试总结

视频标签挖掘需要融合文本、ASR、OCR、视觉、音频和行为信号，通过弱监督、主动学习和多标签模型形成闭环。真正难点是标签体系、长尾、热点更新和不同业务对精确率/召回率的要求。
