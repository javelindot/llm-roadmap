# 机器学习相关：LR和FM，RF和GBDT区别，归一化方法，决策树特征选择

## 答案

这道题包含四组传统机器学习基础，可以分开回答。

### 一、LR 和 FM

#### LR

逻辑回归是线性模型：

```text
y_hat = sigmoid(w0 + sum_i wi * xi)
```

优点：

- 简单、训练和推理快；
- 可解释性强；
- 对高维稀疏特征友好；
- 适合作为强基线。

缺点是只能显式学习线性关系，特征交叉需要人工构造。

#### FM

Factorization Machine 在 LR 基础上增加二阶特征交互：

```text
y_hat = w0
      + sum_i wi * xi
      + sum_i sum_j<i <vi, vj> * xi * xj
```

每个特征有一个低维向量 `vi`，交叉权重由向量内积得到。即使某些特征组合在训练中很稀疏，也能通过共享隐向量学习。

推荐或支付风控中，FM 可以学习：

```text
商户行业 x 国家
支付方式 x 币种
设备类型 x 风险等级
```

### 二、RF 和 GBDT

#### Random Forest

- Bagging 思想；
- 多棵树并行训练；
- 对样本和特征随机采样；
- 最终分类投票、回归平均；
- 主要降低方差，抗过拟合能力较强。

#### GBDT

- Boosting 思想；
- 树按顺序训练；
- 后一棵树拟合前面模型的残差或损失负梯度；
- 最终多棵弱树加权求和；
- 主要降低偏差，通常精度更高，但参数和过拟合更敏感。

| 维度 | RF | GBDT |
|---|---|---|
| 训练方式 | 并行 | 串行 |
| 核心思想 | Bagging | Boosting |
| 树之间关系 | 相对独立 | 后树修正前树 |
| 主要作用 | 降低方差 | 降低偏差 |
| 参数敏感度 | 较低 | 较高 |

### 三、常见归一化方法

**Min-Max**

```text
x' = (x - min) / (max - min)
```

缩放到固定区间，对异常值敏感。

**Standardization**

```text
x' = (x - mean) / std
```

均值为 0、标准差为 1，适合 LR、SVM、神经网络等。

**Robust Scaling**

```text
x' = (x - median) / IQR
```

对异常值更稳健。

**Log Transform**

适合交易金额、访问次数等长尾正数：

```text
x' = log(1 + x)
```

树模型通常不依赖特征尺度，因此一般不要求标准化；距离模型、线性模型和神经网络更需要。

数据归一化参数只能用训练集统计，不能用全量数据，否则会泄漏。

### 四、决策树如何选择特征

决策树选择能最大程度降低不纯度的特征和切分点：

- ID3：信息增益；
- C4.5：信息增益率；
- CART 分类树：Gini impurity reduction；
- CART 回归树：MSE/variance reduction。

以 Gini 为例：

```text
Gini(D) = 1 - sum_k p_k^2
```

选择切分后加权 Gini 最小、也就是不纯度下降最大的特征。

### 面试总结

LR 学线性权重，FM 通过隐向量学习二阶交叉；RF 是并行 Bagging，主要降方差，GBDT 是串行 Boosting，逐步拟合残差；归一化要按模型和数据分布选择；决策树根据不纯度下降选择特征与切分点。
