# 给定一个流式整形输入,求中位数

## 答案

流式数据无法预先排序，可以使用两个堆动态维护中位数：

- `lower`：最大堆，保存较小的一半；
- `upper`：最小堆，保存较大的一半；
- 保证 `len(lower) == len(upper)`，或 `lower` 比 `upper` 多一个元素；
- 保证 `max(lower) <= min(upper)`。

Python 的 `heapq` 只有最小堆，因此在 `lower` 中存相反数。

```python
import heapq


class MedianFinder:
    def __init__(self) -> None:
        self.lower: list[int] = []  # max heap, values are negated
        self.upper: list[int] = []  # min heap

    def add_num(self, num: int) -> None:
        if not self.lower or num <= -self.lower[0]:
            heapq.heappush(self.lower, -num)
        else:
            heapq.heappush(self.upper, num)

        if len(self.lower) > len(self.upper) + 1:
            heapq.heappush(self.upper, -heapq.heappop(self.lower))
        elif len(self.upper) > len(self.lower):
            heapq.heappush(self.lower, -heapq.heappop(self.upper))

    def find_median(self) -> float:
        if not self.lower:
            raise ValueError("median is undefined for an empty stream")

        if len(self.lower) > len(self.upper):
            return float(-self.lower[0])
        return (-self.lower[0] + self.upper[0]) / 2.0
```

例如输入 `5, 2, 10, 4`：

```text
[5]          → 5
[2, 5]       → 3.5
[2, 5, 10]   → 5
[2, 4, 5,10] → 4.5
```

### 正确性不变量

每次插入后通过再平衡保持：

```text
size(lower) = size(upper)
或
size(lower) = size(upper) + 1
```

并且所有 `lower` 元素不大于所有 `upper` 元素。因此：

- 奇数个数时，中位数是 `lower` 堆顶；
- 偶数个数时，中位数是两个堆顶的平均值。

### 复杂度

- 单次插入：`O(log n)`；
- 查询中位数：`O(1)`；
- 空间：`O(n)`。

如果流规模极大且只要求近似中位数，可使用 KLL Sketch、t-digest 等分位数摘要，将空间降到次线性；如果存在滑动窗口并需要删除过期元素，则要使用延迟删除的双堆或带秩的数据结构。
