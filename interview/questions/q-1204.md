# 一面自我介绍，项目深挖，熟悉哪个险种，百万医疗风控怎么做需要哪些步骤，怎么构建风控模型，健康险分为哪些类别，让你做健康险风控模型你会提取哪些特征，重疾险做什么样的特征工程，如何识别风险

## 答案

根据现行《健康保险管理办法》，健康保险包括疾病保险、医疗保险、失能收入损失保险、护理保险和医疗意外保险。百万医疗通常属于费用补偿型医疗保险，风险建模重点不是简单预测“谁会生病”，而是控制逆选择、虚假告知、异常医疗和理赔欺诈，同时保护消费者权益。

### 1. 风控全流程

```text
产品与定价
 → 投保准入与健康告知
 → 等待期和保全
 → 就医与理赔受理
 → 医疗合理性审核
 → 反欺诈调查
 → 赔付/人工复核
 → 申诉与模型反馈
```

不同阶段的标签、可用数据和风险成本不同，不能训练一个总分覆盖所有决策。

### 2. 投保与核保风险

主要识别：

- 既往症、健康告知遗漏与资料矛盾；
- 短期集中投保和逆选择；
- 同一主体多账户、异常设备或代理渠道；
- 身份、职业、地区与保额组合异常；
- 等待期附近的早期出险风险；
- 团险成员或关系网络中的异常聚集。

模型可以做风险分层和人工核保排序，但健康告知、除外责任和承保决定必须依据产品条款、监管要求和人工流程，不能仅凭黑盒分数拒保。

### 3. 百万医疗理赔风控

百万医疗通常具有高保额、免赔额和费用报销属性，理赔侧关注：

- 保单是否有效、等待期和责任范围；
- 发票、病历、处方、检查和出院记录一致性；
- 诊断、治疗项目、住院天数与费用合理性；
- 社保或其他保险结算、免赔额和重复报销；
- 医院、科室、医生、患者和代理人的关系网络；
- 短期高频就医、拆单、异常项目组合；
- OCR 文档篡改、重复票据和跨案件复用；
- 历史理赔、退保、申诉和调查结果。

医疗必要性不能只由通用模型判断，应结合医学规则、专业审核人员和可追溯证据。

### 4. 重疾险特征

重疾险通常按合同约定的疾病状态给付，特征侧更关注：

- 健康告知、既往病史和体检指标；
- 疾病首次诊断时间与等待期；
- 诊断标准、病理/影像/检验材料的一致性；
- 多家医院记录的时间线；
- 家族史等依法合规、与风险直接相关的信息；
- 投保后短期异常就诊；
- 同一资料、设备、联系方式或代理关系的案件聚集；
- 医疗文书真实性和重复使用。

不能把受保护或与保险风险无合理关系的属性作为歧视性代理特征。敏感健康数据应遵循明确目的、最小必要、访问审计和保留期限。

### 5. 模型与规则组合

推荐架构是：

1. 确定性规则处理条款、等待期、金额和资料完整性；
2. LightGBM/XGBoost 处理结构化风险；
3. OCR/NLP/VLM 抽取病历和票据字段；
4. 图模型识别医院、患者、代理和票据关系；
5. 异常检测发现新型模式；
6. 风险分数进入人工审核队列；
7. 专业人员给出最终决定和理由。

LLM 可用于材料摘要、矛盾点提示和调查报告草稿，但不得编造诊断、替代医生判断或直接作出拒赔决定。

### 6. 标签与数据切分

- 投保风险、早期出险、异常医疗和确认欺诈分别建标签；
- 欺诈确认存在长延迟，必须设置成熟窗口；
- 按投保/理赔时间切分，避免未来信息泄漏；
- 同一客户、家庭、医院或团伙不能跨训练测试泄漏；
- 区分“人工调查命中”和“最终确认”，避免把旧策略偏差当真值；
- 使用合法取得、与明确用途相关的数据。

### 7. 评测

类别极不平衡时，不能只看 Accuracy 或 ROC-AUC。应重点看：

- PR-AUC；
- 固定人工审核量下的 Recall/Precision；
- 捕获的风险金额和净节省；
- 误报率、正常案件增加的处理时长；
- 概率校准；
- 不同产品、人群、医院和渠道分桶；
- 申诉、投诉和错误拒绝；
- 概念漂移与团伙策略变化。

上线采用 shadow、灰度和 champion-challenger。高风险模型必须可解释、可审计、可申诉，并保留人工复核。

### 8. 与跨境支付风控的共通与差异

共通能力包括身份核验、关系图谱、异常检测、时间切分、极不平衡学习和人工审核闭环。差异在于保险需要医学与产品条款知识，不能把支付欺诈特征直接迁移为健康风险判断。项目经验可以迁移风控方法论，但领域标签、审核规则和最终决策必须由保险精算、医学与理赔专家共同定义。
