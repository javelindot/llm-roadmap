# 梯度下降是局部最优还是全局最优？

## 答案

梯度下降得到局部最优还是全局最优，取决于目标函数、约束、学习率和优化过程，不能一概而论。

### 1. 凸优化

若目标函数是凸函数，则任意局部最优都是全局最优。对于强凸且光滑的函数，在合适学习率下，梯度下降还可收敛到唯一全局最优点。

典型例子是加入适当凸正则的线性回归、逻辑回归。若函数不可微，可使用次梯度或其他凸优化方法。

### 2. 非凸优化

深度神经网络的损失面通常是非凸的，梯度下降只能保证在一定条件下趋向一阶驻点：

```text
gradient f(x) ≈ 0
```

该点可能是：

- 局部最小值；
- 鞍点；
- 平坦区域；
- 在少数情况下是局部最大值。

因此没有一般性的全局最优保证。SGD 的随机噪声有时有助于逃离尖锐鞍点或局部区域，但也不等于保证找到全局最优。

### 3. 为什么深度学习仍有效

深度网络高维损失面中，大量局部极小值可能具有相近训练损失；真正关心的是验证集泛化和业务指标，而不是证明参数达到数学全局最优。初始化、归一化、残差结构、学习率调度、动量和正则化都会影响最终解。

### 4. 面试总结

凸问题中，在满足条件时梯度下降可以找到全局最优；非凸深度学习中通常只能期望找到较好的驻点或局部解，不能声称保证全局最优。
