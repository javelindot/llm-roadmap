# 还有哪些交叉模型？

## 答案

**一句话**：交叉模型主要用于显式或隐式建模特征交互，比如 FM、FFM、DeepFM、xDeepFM、DCN、PNN、AutoInt 等。

### 常见模型

- FM：二阶特征交叉。
- FFM：按 field 区分隐向量。
- DeepFM：FM + DNN，兼顾低阶和高阶交叉。
- xDeepFM：显式高阶交叉 CIN。
- DCN/DCNv2：Cross Network 显式交叉。
- PNN：用 product layer 建模特征交互。
- AutoInt：用 attention 做特征交互。

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付风控或通道路由里，特征交叉很重要。比如国家和币种组合、商户行业和交易金额组合、设备和 IP 组合，都可能比单个特征更有意义。
>
> 如果是表格风控模型，可以用 DeepFM、DCN 这类模型建模交叉；如果追求可解释，也可以手工构造一些强业务交叉特征，比如“高风险国家 + 新设备 + 大额付款”。

**钩子**：可以引到“FM/DeepFM/DCN、支付风控组合特征、显式交叉和业务解释”。
