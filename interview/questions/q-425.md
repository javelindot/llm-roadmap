# 🔥 介绍一下layernorm和batchnorm的区别？

## 答案

**核心区别**：BatchNorm 沿 Batch 统计每个特征的均值和方差；LayerNorm 在单个样本内部，沿特征维度做归一化。

### 1. 归一化维度

假设输入形状是 `[batch, seq, hidden]`：

- **LayerNorm**：对每个 Token 的 `hidden` 维归一化，不依赖其他样本。
- **BatchNorm**：通常对同一通道收集 Batch 及空间位置的统计量。

### 2. 对比

| 维度 | BatchNorm | LayerNorm |
|------|-----------|-----------|
| 统计范围 | Batch 内同一特征 | 单样本内部特征 |
| 是否依赖 Batch Size | 是 | 否 |
| 训练/推理是否一致 | 不完全一致，推理用移动统计量 | 一致 |
| 变长序列适配 | 较麻烦 | 自然 |
| 常见领域 | CNN、视觉模型 | Transformer、RNN、LLM |

### 3. 为什么 Transformer 更常用 LayerNorm

- NLP 的句子长度和 Batch 组成经常变化。
- 自回归推理时 Batch 可能很小，甚至只有 1。
- Token 表示的隐藏维语义更适合在样本内部归一化。
- 不依赖全局 Batch 统计，训练和在线推理行为一致。

### 4. 补充：RMSNorm

很多现代 LLM 使用 RMSNorm。它不减均值，只按均方根缩放：

```text
RMSNorm(x) = x / sqrt(mean(x²) + eps) * g
```

计算更简单，实践中通常能保持较好的训练稳定性。

**面试总结**：BN 更适合 Batch 统计稳定的视觉特征；LN 与 Batch 无关，更适合变长序列和自回归 Transformer。
