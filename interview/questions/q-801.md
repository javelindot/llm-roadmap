# 如何做 Dropout

## 答案

Dropout 在训练时随机屏蔽神经元，减少特征之间的共适应，属于随机正则化。

### Inverted Dropout

设丢弃概率为 \(p\)，保留率 \(q=1-p\)，掩码：

\[
m_i\sim\operatorname{Bernoulli}(q)
\]

训练输出：

\[
y_i=\frac{m_i}{q}x_i
\]

因为：

\[
\mathbb E[y_i]=x_i
\]

所以推理时直接使用 \(y=x\)，无需再缩放。这是主流框架采用的 inverted dropout。

```python
dropout = torch.nn.Dropout(p=0.1)
model.train()  # 随机丢弃并按 1/(1-p) 缩放
model.eval()   # 不丢弃
```

### 常见位置

- Embedding 或位置编码之后；
- Attention 权重或 Attention 输出；
- FFN 激活与残差分支；
- 分类头之前；
- LoRA 分支输入。

### 注意事项

- 验证和推理必须 `eval()`；
- 小数据可适当增加，超大规模预训练未必需要较高 dropout；
- RNN 可用 variational/locked dropout，避免每个时间步掩码剧烈变化；
- DropPath/Stochastic Depth 是随机丢残差分支，不等同普通 dropout；
- 随机性会影响可复现与分布式训练，应保存 seed 和 RNG state。

### 面试总结

训练时用 Bernoulli mask 丢弃并除以保留率，使期望不变；推理时关闭 Dropout。它缓解过拟合但不是越大越好，应按数据规模、插入位置和验证集选择。
