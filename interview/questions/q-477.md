# bert详解，有什么缺点，bert有哪些改进模型；

## 答案

### 1. BERT 的结构与训练

BERT 是 Transformer Encoder-only 模型，核心特点是双向 Self-Attention。输入表示由三部分相加：

```text
Token Embedding + Position Embedding + Segment Embedding
```

序列前加入 `[CLS]` 用于句级任务，句子之间可用 `[SEP]` 分隔。原始 BERT 主要使用：

- **MLM**：随机遮盖部分 Token，预测被遮盖内容。
- **NSP**：判断两个句子是否相邻。

预训练后通过 Fine-tuning 适配分类、NER、问答和文本匹配。

### 2. 优点

- 双向上下文让每个 Token 同时利用左右信息。
- 相比静态 Word2Vec，同一词在不同语境中有不同表示。
- 相比 RNN，Transformer 训练可并行，长距离依赖路径更短。
- 大规模预训练后，少量标注数据即可迁移到下游任务。

### 3. 缺点

- MLM 中 `[MASK]` 在真实下游输入里不存在，存在预训练—微调差异。
- NSP 的收益有限，负样本构造较简单。
- Self-Attention 对长序列的时间和空间成本为平方级。
- Encoder-only 不擅长自回归长文本生成。
- 原始位置长度有限，超长文本需要改造。
- 模型参数和推理成本仍高于传统轻量模型。

### 4. 代表性改进

| 模型 | 主要改进 |
|------|----------|
| RoBERTa | 去掉 NSP，动态 Mask，更大数据与更充分训练 |
| ALBERT | 参数共享、Embedding 分解，降低参数量 |
| ELECTRA | 用替换 Token 检测提高预训练样本效率 |
| DeBERTa | 内容与位置解耦的注意力表示 |
| DistilBERT | 蒸馏压缩，提升推理速度 |
| SpanBERT | 按 Span Mask，增强跨度表示 |
| Longformer / BigBird | 稀疏注意力支持更长文本 |
| MacBERT | 更自然的中文掩码纠错式预训练 |

**面试总结**：BERT 的突破在于“双向上下文预训练 + 统一 Fine-tuning”，后续改进主要围绕训练效率、预训练任务、位置建模和长文本能力。
