# ResNet的核心思想是什么？

## 答案

ResNet 的核心是残差学习和恒等捷径连接。不是让网络直接学习目标映射 \(H(x)\)，而是学习：

\[
F(x)=H(x)-x
\]

输出：

\[
y=F(x)+x
\]

### 为什么有效

- 恒等 shortcut 提供直接的信息和梯度通道；
- 若新增层暂时无用，只需学到 \(F(x)\approx0\)，比重新拟合恒等映射容易；
- 缓解深层网络的 degradation problem：层数增加后训练误差反而变高；
- 支持训练几十到上百层网络。

反向传播：

\[
\frac{\partial y}{\partial x}
=I+\frac{\partial F}{\partial x}
\]

即使残差分支梯度较小，仍有恒等项传播。

### 两类 Block

- BasicBlock：两个 \(3\times3\) 卷积，常用于 ResNet-18/34；
- Bottleneck：\(1\times1\) 降维、\(3\times3\)、\(1\times1\) 升维，常用于 ResNet-50/101/152。

输入输出尺寸不同使用 projection shortcut，如 \(1\times1\) 卷积匹配维度。

[ResNet 原论文](https://arxiv.org/abs/1512.03385)解决的是深层优化退化，不等同于彻底消除梯度问题。残差思想后来成为 Transformer 和扩散模型的基础组件。

### 面试总结

ResNet 通过 \(y=x+F(x)\) 学残差而非完整映射，恒等 shortcut 让信息与梯度直接传播，解决深层网络难优化和退化问题。
