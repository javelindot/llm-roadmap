# 给定两个单词一个转换成另一个，只用删除，插入和替换，返回最小操作数，以及打印每一步操作

## 答案

这是编辑距离。定义 `dp[i][j]` 为 `word1[:i]` 转换到 `word2[:j]` 的最少操作数。

### 状态转移

边界：

$$
dp[i][0]=i,\qquad dp[0][j]=j
$$

若末尾字符相同：

$$
dp[i][j]=dp[i-1][j-1]
$$

否则：

$$
dp[i][j]=1+\min
\begin{cases}
dp[i-1][j] & 删除\\
dp[i][j-1] & 插入\\
dp[i-1][j-1] & 替换
\end{cases}
$$

### Python 实现与路径回溯

```python
def edit_distance_with_ops(a, b):
    m, n = len(a), len(b)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if a[i - 1] == b[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = 1 + min(
                    dp[i - 1][j],
                    dp[i][j - 1],
                    dp[i - 1][j - 1],
                )

    ops, i, j = [], m, n
    while i > 0 or j > 0:
        if i > 0 and j > 0 and a[i - 1] == b[j - 1]:
            i, j = i - 1, j - 1
        elif i > 0 and j > 0 and dp[i][j] == dp[i - 1][j - 1] + 1:
            ops.append(("replace", i - 1, a[i - 1], b[j - 1]))
            i, j = i - 1, j - 1
        elif i > 0 and dp[i][j] == dp[i - 1][j] + 1:
            ops.append(("delete", i - 1, a[i - 1]))
            i -= 1
        else:
            ops.append(("insert", i, b[j - 1]))
            j -= 1

    return dp[m][n], ops[::-1]
```

时间与用于回溯的空间都是 $O(mn)$。若只求距离可滚动数组降到 $O(\min(m,n))$，但完整打印路径通常仍需保存父指针或 DP 表。最优路径可能不唯一，上述优先级只返回其中一条。
