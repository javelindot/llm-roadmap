# 下一个排序

## 答案

题干“下一个排序”信息不足，常见对应题目是 LeetCode 31“下一个排列”。以下按这个题意回答：给定一个整数数组，原地变为字典序中下一个更大的排列；如果不存在，则变为最小排列。

### 1. 思路

以：

```text
1 2 7 4 3 1
```

为例。

1. 从右向左找第一个 `nums[i] < nums[i+1]`，这里是 `2`；
2. 从右向左找第一个大于 `nums[i]` 的数，这里是 `3`；
3. 交换 `2` 和 `3`；
4. 将 `i+1` 后面的降序区间反转为升序。

结果：

```text
1 3 1 2 4 7
```

### 2. Python 实现

```python
def next_permutation(nums):
    n = len(nums)

    # 1. 找到第一个下降位置
    i = n - 2
    while i >= 0 and nums[i] >= nums[i + 1]:
        i -= 1

    # 2. 存在更大的排列
    if i >= 0:
        j = n - 1
        while nums[j] <= nums[i]:
            j -= 1
        nums[i], nums[j] = nums[j], nums[i]

    # 3. 反转后缀
    left, right = i + 1, n - 1
    while left < right:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
        right -= 1
```

### 3. 为什么正确

右侧最长非递增后缀已经是当前前缀下的最大排列。要得到“刚好更大”的排列：

- 尽量在靠右的位置增大一个数；
- 用后缀中刚好大于它的最小值交换；
- 再把后缀变成最小的升序排列。

### 4. 复杂度

```text
时间复杂度：O(n)
空间复杂度：O(1)
```

### 5. 面试总结

如果原题确实是“下一个排列”，核心是“从右找转折点、交换刚好更大的数、反转后缀”。如果“下一个排序”指的是其他题，必须先补充完整输入输出，不能直接套用该答案。
