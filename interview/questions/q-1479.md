# dp和贪心的区别

## 答案

动态规划和贪心都利用问题结构减少搜索，但决策方式不同：

- 贪心每一步选择当前看来最优的方案，通常不回退；
- DP 保存子问题最优解，综合比较后得到全局最优。

### 1. 动态规划

适用条件：

- 最优子结构；
- 重叠子问题；
- 状态和转移可定义。

步骤：

```text
定义 dp 状态
-> 写转移方程
-> 初始化
-> 确定遍历顺序
-> 返回答案
```

例如 0/1 背包：

```text
dp[i][capacity]
```

每个物品要比较选或不选，局部最大价值选择不一定全局最优。

### 2. 贪心

适用条件：

- 贪心选择性质；
- 局部最优可导出全局最优；
- 通常需要交换论证、归纳或拟阵等证明。

例如无权区间调度：每次选择结束时间最早的区间，可获得最多互不重叠区间。

### 3. 对比

| 维度 | DP | 贪心 |
|---|---|---|
| 决策 | 比较多个子状态 | 当前局部最优 |
| 是否保存状态 | 是 | 通常少 |
| 是否回看其他选择 | 会综合 | 通常不会 |
| 正确性条件 | 最优子结构+重叠 | 贪心选择性质 |
| 复杂度 | 通常更高 | 通常更低 |

### 4. 反例

硬币 `[1,3,4]` 凑 6：

- 贪心先选 4，再选 1、1，共 3 枚；
- 最优是 3+3，共 2 枚。

说明没有证明时不能凭“每次选最大”使用贪心。

### 5. 如何判断

- 能否证明某个局部选择总可出现在最优解中？尝试贪心；
- 当前选择会影响未来，且需要比较多个历史方案？考虑 DP；
- 状态是否能压缩；
- 是否存在简单反例。

### 6. 面试总结

DP 枚举并缓存必要状态，用更高时间/空间换全局最优；贪心只保留局部最优，效率高但必须证明。两者都依赖最优子结构，区别在是否需要保留多个选择。
