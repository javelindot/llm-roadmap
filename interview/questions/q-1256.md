# 二叉搜索树的插入、删除操作及构建方法

## 答案

二叉搜索树 BST 满足：对每个节点，左子树所有键小于该节点，右子树所有键大于该节点。若允许重复键，需要提前约定放左、放右或在节点中记录计数；以下实现不插入重复值。

### 1. 插入

从根开始比较：

- 小于当前键，进入左子树；
- 大于当前键，进入右子树；
- 遇到空位置时创建新节点。

### 2. 删除

删除节点分三种情况：

1. 没有子节点：直接删除；
2. 只有一个子节点：用唯一子节点替代；
3. 有两个子节点：找到右子树最小节点，即中序后继，用其键替换当前节点，再从右子树删除该后继。

也可以使用左子树最大节点作为前驱，原则一致。

### 3. Python 实现

```python
from dataclasses import dataclass


@dataclass
class Node:
    key: int
    left: "Node | None" = None
    right: "Node | None" = None


def insert(root: Node | None, key: int) -> Node:
    if root is None:
        return Node(key)
    if key < root.key:
        root.left = insert(root.left, key)
    elif key > root.key:
        root.right = insert(root.right, key)
    return root


def find_min(root: Node) -> Node:
    current = root
    while current.left is not None:
        current = current.left
    return current


def delete(root: Node | None, key: int) -> Node | None:
    if root is None:
        return None

    if key < root.key:
        root.left = delete(root.left, key)
    elif key > root.key:
        root.right = delete(root.right, key)
    else:
        if root.left is None:
            return root.right
        if root.right is None:
            return root.left

        successor = find_min(root.right)
        root.key = successor.key
        root.right = delete(root.right, successor.key)

    return root


def build_bst(values: list[int]) -> Node | None:
    root = None
    for value in values:
        root = insert(root, value)
    return root


def inorder(root: Node | None) -> list[int]:
    if root is None:
        return []
    return inorder(root.left) + [root.key] + inorder(root.right)
```

### 4. 测试

```python
root = build_bst([5, 3, 7, 2, 4, 6, 8])
assert inorder(root) == [2, 3, 4, 5, 6, 7, 8]

root = delete(root, 2)
assert inorder(root) == [3, 4, 5, 6, 7, 8]

root = delete(root, 7)
assert inorder(root) == [3, 4, 5, 6, 8]

root = delete(root, 5)
assert inorder(root) == [3, 4, 6, 8]
```

### 5. 复杂度

插入、查找和删除都是 `O(h)`，其中 `h` 是树高：

- 平衡时 `h = O(log n)`；
- 按有序数组依次插入时会退化成链表，`h = O(n)`。

若需要稳定的 `O(log n)`，应使用 AVL 树、红黑树或随机 Treap。对已排序数组构建平衡 BST，可以每次选择中点作为根，整体 `O(n)` 完成，而不是顺序插入。
