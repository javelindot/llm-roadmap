# 4.逻辑回归

## 答案

**一句话**：逻辑回归是用于二分类的线性模型，通过 sigmoid 把线性打分映射成概率，用交叉熵训练。

### 原理

```text
z = w^T x + b
p = sigmoid(z)
```

输出 `p` 表示正类概率。训练通常用二元交叉熵。逻辑回归可解释性强、训练快，适合做 baseline。

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付风控里，逻辑回归可以作为可解释 baseline，比如预测交易是否高风险。每个特征权重能解释方向，比如高风险国家、异常设备、大额交易会提高风险概率。
>
> 线上复杂模型可能用 GBDT 或深度模型，但逻辑回归仍然适合做基线、兜底和解释。

**钩子**：可以引到“sigmoid、BCE、可解释 baseline、风控特征权重”。
