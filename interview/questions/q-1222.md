# 多模态的DPO的训练是如何做的，如果构造数据？

## 答案

多模态 DPO 把文本 DPO 的条件输入从 Prompt 扩展为图像、文档或视频等模态与文本指令。典型样本是：

```text
(visual_input, prompt, chosen_response, rejected_response)
```

训练目标要求在同一视觉条件和指令下，chosen 相对 reference model 的概率提升幅度大于 rejected。

### 1. 训练流程

1. 选择已经完成多模态预训练和 SFT 的 VLM 作为 policy 初始化。
2. 冻结同一 SFT checkpoint 作为 reference model。
3. 构造图文条件下的偏好对，并完成去重、质量与泄漏检查。
4. 计算 policy/reference 对 chosen、rejected 的序列 log-prob。
5. 使用 DPO loss 更新语言模型、视觉投影层或 LoRA Adapter；视觉 Encoder 是否更新取决于任务偏移和数据规模。
6. 在冻结的视觉 grounding、幻觉、安全和业务测试集上评测。

条件形式可写为：

```text
L = -log sigmoid(
    beta * (
        log pi_theta(y_w | I, x) - log pi_ref(y_w | I, x)
        - log pi_theta(y_l | I, x) + log pi_ref(y_l | I, x)
    )
)
```

其中 `I` 是视觉输入，`x` 是文本指令。训练时通常只对回答 token 计算 loss，不把 Prompt 或视觉占位 token 当作监督答案。

### 2. 偏好数据来源

**人工标注**

- 对同一图片和问题采样多个回答；
- 标注者根据事实性、视觉证据、完整性和安全性排序；
- 专业文档由懂业务的标注者复核。

**模型生成加人工筛选**

- 强模型生成候选、批改或定位错误；
- 人工抽检高风险、低置信度和模型分歧样本；
- 不能只依赖同一个模型既生成又评判。

**可验证任务**

- OCR 字段与标注值比较；
- 表格、票据、证件字段做结构化校验；
- 目标检测、区域引用或计数可由标注框验证；
- 合成图片可精确控制对象、属性和位置。

**硬负样本**

- 把正确实体替换成同页相似实体；
- 修改数量、日期、币种、地址或否定关系；
- 将另一张图的正确答案作为当前图的错误答案；
- 对图片做局部遮挡、裁剪或替换，构造 image-contrast preference。

硬负样本应自然且只改变关键事实。如果 rejected 同时更短、语法更差或带固定提示词，模型会学习表面风格而不是视觉 grounding。

### 3. 多模态 DPO 的关键风险

标准 DPO 可能出现 **unconditional preference**：即使移除图片，模型仍能根据语言风格判断 chosen，训练后并没有更关注视觉输入。可以通过以下方式发现和缓解：

- 比较带图、去图、错图三种条件下的 log-prob 和准确率；
- 保持 chosen/rejected 长度、语气和格式尽量一致；
- 加入同回答、不同图像的 image preference；
- 使用视觉对比负样本或 MDPO、V-DPO 类条件目标；
- 对关键答案要求区域、页码或证据引用；
- 单独评测 object hallucination 和跨模态冲突。

还要防止 OCR 文本泄漏标签、同一文档页面跨训练/测试集、图片近重复，以及 rejected 由低质量模板批量生成。

### 4. 跨境支付数据构造

KYC/KYB、付款凭证和后台截图可形成高价值多模态任务，但必须先脱敏并满足数据授权：

- 营业执照：公司名、注册号、有效期和签发地；
- 银行回单：金额、币种、日期、付款方与收款方；
- 商户后台：交易状态、错误码和操作入口；
- 多页材料：字段跨页一致性与缺失材料判断。

chosen 必须与可见证据和 OCR 结构化结果一致；rejected 可以是易混字段、错误币种、过期日期、错页实体或无证据推断。完整账号、证件号和个人信息应遮盖或使用合成数据。

模型只能做材料抽取、解释和风险提示。开户通过、制裁命中处置、资金放行等决定仍需规则引擎和人工审批。

### 5. 评测指标

- 字段级 exact match、F1 和数值/日期规范化准确率；
- 图文一致性、区域 grounding 和证据引用正确率；
- object/attribute/relation hallucination；
- 去图、错图和遮挡后的敏感性；
- 多语言 OCR、低清、旋转和复杂表格鲁棒性；
- PII 泄露、越权建议和拒答正确率；
- 与 SFT-only、普通 DPO 和多模态条件 DPO 的消融对比。
