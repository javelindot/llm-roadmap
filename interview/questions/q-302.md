# FastAPI怎么加载环境变量？WebSocket怎么实现实时调用？异步隔离？

## 答案

答：load_dotenv()加载.env文件；
@app.websocket(“/ws”) async def func；
FastAPI底层是Starlette+Asyncio，自带请求级隔离。

### 补充说明

环境变量一般用 `python-dotenv` 的 `load_dotenv()` 加载 `.env`，再用 `os.getenv()` 或 Pydantic Settings 读取。生产环境更推荐通过容器环境变量、K8s Secret、配置中心注入，避免把敏感配置写进代码。

WebSocket 在 FastAPI 里通过 `@app.websocket("/ws")` 实现，连接后 `accept()`，循环 `receive_text()` 和 `send_text()`。如果要实时调用大模型，可以边收到模型 streaming token，边通过 WebSocket 推给前端。

异步隔离要注意：I/O 型任务可以用 async；CPU 密集或阻塞任务不能直接阻塞事件循环，要放线程池、进程池或任务队列。

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付客服项目里，FastAPI 可以用来封装模型服务或 RAG 服务。环境变量里会放模型地址、向量库地址、API key、超时时间，但这些敏感信息生产上要走 Secret 或配置中心。
>
> WebSocket 适合做流式客服回复。比如用户问付款为什么失败，后端一边查订单和知识库，一边把模型生成的内容流式推给前端，用户不需要等完整答案结束。
>
> 异步隔离我会特别注意。查询订单、调用模型、访问向量库都是 I/O，可以 async；但 PDF OCR、图片解析、embedding 批处理这类重任务要放队列，不能阻塞实时客服请求。

**钩子**：可以引到“Pydantic Settings、WebSocket streaming、阻塞任务隔离、生产 Secret 管理”。
