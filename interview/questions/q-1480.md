# 手撕:正则表达式匹配

## 答案

通常指 LeetCode 10：实现字符串 `s` 与模式 `p` 的完整匹配，其中：

- `.` 匹配任意单个字符；
- `*` 匹配前一个元素零次或多次；
- 要求匹配整个字符串。

### 1. DP 定义

```text
dp[i][j] =
s 的前 i 个字符
是否与 p 的前 j 个字符完全匹配
```

初始化：

```text
dp[0][0] = True
```

空字符串可被 `a*b*c*` 匹配：

```text
dp[0][j] = dp[0][j-2], if p[j-1] == '*'
```

### 2. 转移

若当前模式不是 `*`：

```text
match(s[i-1], p[j-1]) and dp[i-1][j-1]
```

若当前模式是 `*`：

**匹配零次**

```text
dp[i][j-2]
```

**匹配一次或多次**

```text
match(s[i-1], p[j-2]) and dp[i-1][j]
```

### 3. Python 实现

```python
def is_match(s, p):
    m = len(s)
    n = len(p)

    dp = [[False] * (n + 1) for _ in range(m + 1)]
    dp[0][0] = True

    for j in range(2, n + 1):
        if p[j - 1] == "*":
            dp[0][j] = dp[0][j - 2]

    def matches(i, j):
        if i == 0:
            return False
        return p[j - 1] == "." or s[i - 1] == p[j - 1]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if p[j - 1] != "*":
                dp[i][j] = (
                    matches(i, j)
                    and dp[i - 1][j - 1]
                )
            elif j >= 2:
                dp[i][j] = dp[i][j - 2]

                if matches(i, j - 1):
                    dp[i][j] = (
                        dp[i][j] or dp[i - 1][j]
                    )

    return dp[m][n]
```

### 4. 复杂度

```text
时间复杂度：O(mn)
空间复杂度：O(mn)
```

可使用滚动数组优化空间，但二维 DP 更易在面试中写对。

### 5. 易错点

- `*` 作用于前一个模式字符；
- `dp[i-1][j]` 表示继续用同一个 `x*`；
- 完整匹配不是子串搜索；
- 空字符串初始化；
- 模式以 `*` 开头通常视为非法；
- `.*` 可以匹配任意长度字符串。

### 6. 面试总结

关键是把 `x*` 拆成零次 `dp[i][j-2]` 和多次 `dp[i-1][j]`。先定义 DP 语义和空串初始化，再写代码不容易错。
