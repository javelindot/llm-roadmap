# 卷积层有哪些基本参数？

## 答案

**一句话**：卷积层的基本参数包括卷积核大小、步幅、填充、通道数、卷积核个数、空洞率和分组数。

### 核心回答
- **kernel size**：卷积核大小，如 3x3、5x5。
- **stride**：步幅，决定滑动间隔和输出尺寸。
- **padding**：填充，控制边界信息和输出尺寸。
- **in_channels/out_channels**：输入/输出通道数。
- **dilation**：空洞卷积，扩大感受野。
- **groups**：分组卷积，减少参数和计算。

输出尺寸常见公式：

```text
out = floor((in + 2 * padding - dilation * (kernel - 1) - 1) / stride + 1)
```

### 1. 每个参数影响什么

- kernel size 决定一次看多大局部区域。
- stride 决定滑动步长，影响输出尺寸和下采样强度。
- padding 决定边界信息是否保留。
- out_channels 决定输出特征数量。
- dilation 可以在不增加参数的情况下扩大感受野。
- groups 可以减少计算，比如 depthwise convolution。

### 2. 参数怎么影响效果

小卷积核适合提取细粒度纹理，大卷积核或空洞卷积适合看更大范围。stride 太大可能丢细节，padding 不合适会影响边界信息。

所以卷积参数本质是在感受野、计算量、细节保留之间做平衡。

### 3. 面试总结

回答卷积参数时，最好不要只背名词，要说出它们如何影响输出尺寸、感受野、计算量和特征细节。

### 4. 结合跨境支付项目怎么说（面试口径）

> 如果项目里做证照图片处理，这些参数会影响模型看图的粒度。比如营业执照 OCR 前先做图像质量判断，小卷积核适合看局部边缘和文字纹理，大感受野有助于看整体版式。
>
> 支付场景里图片审核通常不是核心资金链路，但它能辅助 KYB，发现材料模糊、截图伪造、图文不一致这类问题。
>
> 我会特别注意不要因为下采样太强丢掉小字、印章、证件号这类关键细节，所以证照类任务往往要兼顾高分辨率和推理成本。

**钩子**：卷积参数最终影响感受野和计算量，证照审核要兼顾细节和版面。
