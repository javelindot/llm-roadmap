# 什么是归一化

## 答案

“归一化”在面试中可能指输入特征缩放，也可能指神经网络中的 Normalization Layer，最好先区分语境。

### 1. 数据归一化与标准化

Min-Max 归一化：

```text
x' = (x - min) / (max - min)
```

通常映射到 `[0, 1]`，但对异常值敏感。

Z-score 标准化：

```text
x' = (x - mean) / std
```

使数据均值接近 0、标准差接近 1。长尾金额还可先做 `log1p`，或使用 median/IQR 的 Robust Scaling。

所有统计量只能在训练集拟合，再原样应用到验证集和线上，否则会数据泄漏。树模型通常不敏感，但线性模型、距离模型和神经网络更依赖合理尺度。

### 2. 神经网络归一化

- **BatchNorm**：按 batch 统计，CNN 常用；小 batch 和训练/推理统计不一致是问题；
- **LayerNorm**：按单样本隐藏维归一化，Transformer 常用；
- **RMSNorm**：只按均方根缩放，不减均值，现代 LLM 常见；
- **GroupNorm**：按通道组归一化，适合小 batch 视觉任务；
- **Pre-Norm/Post-Norm**：决定归一化放在残差子层之前或之后，影响深层网络训练稳定性。

这些层通常有可学习缩放参数，并不只是把数值限制到固定区间。

### 3. 作用

- 缓解不同特征尺度带来的优化困难；
- 改善梯度传播和数值稳定性；
- 允许更稳定的学习率；
- 降低某些分布漂移，但不能消除真实业务漂移。

### 4. 跨境支付场景

不同币种金额不能直接放在一起归一化。应先按交易时点汇率转换到统一计价尺度，或保留币种并按币种/国家分组处理，再对长尾金额做 `log1p` 和 robust scaling。

汇率、统计窗口和特征版本必须时间一致，不能用未来汇率或全量数据统计量。

### 5. 面试总结

输入归一化处理特征尺度，网络归一化处理层内激活分布和优化稳定性。回答时要能写出 Min-Max、Z-score，并说明 BatchNorm、LayerNorm、RMSNorm 的统计维度和适用场景。
