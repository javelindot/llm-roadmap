# 什么是对比学习(Contrastive Learning)？InfoNCE loss 的公式和作用？

## 答案

对比学习通过比较样本关系学习表示：语义相同或相关的正样本在 embedding 空间接近，不相关的负样本远离。它常用于自监督表示、跨模态对齐和检索。

### InfoNCE

给定 anchor $q$、正样本 $k^+$ 和负样本集合：

$$
\mathcal{L}=
-\log\frac{\exp(sim(q,k^+)/\tau)}
{\exp(sim(q,k^+)/\tau)+
\sum_{k^-}\exp(sim(q,k^-)/\tau)}
$$

`sim` 常为 L2 归一化后的点积，即 cosine similarity；$\tau$ 是 temperature。

它可以看成一个分类问题：在候选 key 中识别与 query 匹配的正样本。最小化 loss 会增大正对相似度、减小负对相似度。InfoNCE 也与互信息下界有关，但实际效果更依赖数据增强和负样本质量。

### Temperature 的作用

$\tau$ 小时 softmax 更尖锐，模型更关注难负例，但噪声和 false negative 的影响也更大；$\tau$ 大时分布更平滑，梯度区分度减弱。可固定或学习，并在验证集调节。

### 数据与局限

正样本增强必须保持语义，负样本不应实际同义。batch 内负例、memory queue 和 hard negative mining 可增加难度；大 batch 也会增加 false negative。

没有显式负样本的方法如 BYOL/SimSiam 通过 stop-gradient、非对称 predictor 等避免表示坍缩。评估应看检索 Recall@K、线性探针和下游任务，不能只看对比 loss。
