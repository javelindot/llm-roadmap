# 手撕代码:有序链表去重

## 答案

“有序链表去重”有两种常见题意，面试时必须确认：

1. 重复值只保留一个，LeetCode 83；
2. 所有出现重复的值都删除，LeetCode 82。

### 1. 重复只保留一个

因为链表有序，相同值一定连续：

```python
def delete_duplicates(head):
    cur = head

    while cur and cur.next:
        if cur.val == cur.next.val:
            cur.next = cur.next.next
        else:
            cur = cur.next

    return head
```

复杂度：

```text
时间 O(n)
空间 O(1)
```

### 2. 删除所有重复值

例如：

```text
1 -> 2 -> 2 -> 3 -> 4 -> 4
```

结果：

```text
1 -> 3
```

使用 dummy 处理头部重复：

```python
def delete_all_duplicates(head):
    dummy = ListNode(0, head)
    prev = dummy
    cur = head

    while cur:
        duplicated = False

        while cur.next and cur.val == cur.next.val:
            duplicated = True
            cur = cur.next

        if duplicated:
            prev.next = cur.next
        else:
            prev = prev.next

        cur = cur.next

    return dummy.next
```

### 3. 易错点

- 空链表；
- 单节点；
- 头部连续重复；
- 尾部重复；
- 全部节点相同；
- 删除节点后不要错误移动 `prev`；
- 题目要求重连节点，不要只改值。

### 4. 面试总结

有序性使重复节点连续，所以一次遍历即可。保留一个时直接跳过后继；删除全部重复时使用 dummy 和 `prev` 跨过整段重复值。
