# self-attention求内积时为啥除以根号d

## 答案

Self-Attention 的点积除以 `sqrt(d_k)`，是为了让不同 head dimension 下 logits 的方差保持在稳定尺度，避免 softmax 过早饱和、梯度变小。

### 1. 方差推导

假设 Query 和 Key 每个维度独立、均值为 0、方差为 1：

```text
q · k = sum_(i=1)^d q_i k_i
```

每项 `q_i k_i` 的均值约为 0、方差约为 1，因此：

```text
Var(q · k) ≈ d
Std(q · k) ≈ sqrt(d)
```

随着 `d` 增大，点积绝对值变大。除以 `sqrt(d)` 后：

```text
Var((q · k) / sqrt(d)) ≈ 1
```

使 logits 尺度不随维度线性扩大。

### 2. 不缩放会怎样

大 logits 进入 softmax 后，概率接近 one-hot：

```text
softmax([20, 0, -20]) ≈ [1, 0, 0]
```

非最大位置梯度很小，Attention 过早变得极尖锐，训练不稳定。缩放相当于固定的 temperature，保留更健康的梯度。

### 3. 为什么不是除以 d

点积的标准差增长为 `sqrt(d)`，不是 `d`。除以 `d` 会让方差变为 `1/d`，维度越大 logits 越小，softmax 过于均匀。

### 4. 前提与现代实现

真实 Q/K 不完全独立同分布，但初始化、归一化和投影使该近似具有工程意义。某些模型还使用：

- 可学习 temperature；
- QK normalization；
- logit clipping/softcapping；
- 其他 Attention 参数化。

但标准 scaled dot-product attention 仍使用：

```text
softmax(QK^T / sqrt(d_k))
```

### 5. 面试总结

点积由 `d` 项相加，方差约为 `d`、标准差约为 `sqrt(d)`；除以 `sqrt(d)` 将 logits 恢复到 `O(1)` 尺度，防止 softmax 饱和并稳定梯度。
