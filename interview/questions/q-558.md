# 如果要做电商agent，你会选择哪些模态的信息作为输入？比如文本评论、图像、视频、购买记录？

## 答案

**一句话**：电商 Agent 应该融合文本、图像、视频、行为、交易和用户反馈等多模态信息，不同任务选择不同输入。

### 电商输入模态

- 文本：标题、评论、客服对话、商品描述。
- 图像：商品图、详情图、买家秀。
- 视频：直播、短视频、讲解。
- 行为：点击、收藏、加购、购买、停留。
- 交易：价格、退款、物流、售后。
- 用户画像：偏好、地域、消费能力。

### 结合跨境支付项目怎么说（面试口径）

> 如果迁移到跨境支付 Agent，我会选择文本、结构化交易、附件图片和行为日志四类输入。文本包括客服对话和通道规则；结构化数据包括订单、金额、币种、状态；图片包括 KYC 材料和银行回单；行为日志包括登录设备、IP、历史交易。
>
> 不同任务输入不同。客服解释更依赖文本和订单状态；风控分析更依赖交易序列、设备和图谱关系；KYC 审核更依赖证件图片和 OCR。

**钩子**：可以引到“多模态按任务选择、结构化数据 + 文本 + 图片 + 行为日志”。
