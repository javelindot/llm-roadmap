# softmax公式

## 答案

**一句话**：Softmax 把一组实数转换成概率分布，每个输出为指数值除以所有指数值之和。

### 公式

```text
softmax(x_i) = exp(x_i) / sum_j exp(x_j)
```

输出值都大于 0，且总和为 1。为了数值稳定，实际计算常减去最大值：

```text
softmax(x_i) = exp(x_i - max(x)) / sum_j exp(x_j - max(x))
```

### 结合跨境支付项目怎么说（面试口径）

> 在跨境支付意图分类里，softmax 可以把模型输出转成各类概率，比如付款失败、退款、KYC、风控拦截。概率最高的是预测类别，但如果最高概率也很低，就应该追问或转人工。
>
> 在 Attention 里，softmax 也用于把相似度分数变成注意力权重。

**钩子**：可以引到“分类概率、数值稳定、attention 权重、低置信度转人工”。
