# 你用的模型有多少层

## 答案

必须根据实际模型配置回答，不要凭参数量猜层数。可以查看 `config.json` 中的 `num_hidden_layers`，或代码中的：

```python
num_layers = model.config.num_hidden_layers
```

### 回答模板

> 使用的是[模型完整名称与版本]，包含[层数]个 Transformer Block，隐藏维是[数值]，Attention head 为[数值]，KV head 为[数值]，FFN 中间维为[数值]，总参数约[数值]。我微调的是[目标模块]，不是把所有层都解冻。

还要明确“层”口径：

- 通常指重复的 Transformer blocks；
- 不把 embedding、最终 Norm 和 LM head 算作 block；
- Encoder-Decoder 应分别报告 encoder/decoder 层；
- MoE 的专家数不是层数；
- LoRA rank 也不是网络层数。

面试官继续追问时，通常是在验证是否真正使用过模型，应能说明层数如何影响参数、激活、pipeline 分段和推理延迟。

### 面试总结

从模型配置读取真实 `num_hidden_layers`，并一起说明 hidden size、head、KV head 和 FFN 维度。模型层数是架构事实，不应靠记忆或参数量近似猜测。
