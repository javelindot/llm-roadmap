# 总结一下八股：强化学习（目前每个面试官都会问），L1 L2正则化，LoRA原理，其他PEFT方法，SFT和DPO的区别，SFT和DPO训练心得，Transformer结构，位置编码，长度外推，优化器，flash attention，分类为什么用cse而不是mse，BERT系列训练流程与各种变体，AUC指标，最近读过的论文。

## 答案

这是一份复习范围清单，适合按“能定义、能推导、能实现、能结合项目”四层准备，而不是逐个背一句话。

### 模型与训练主线

- Transformer：MHA、FFN、残差、Pre/Post-Norm，复杂度与 KV Cache。
- 位置编码：绝对位置、RoPE、ALiBi，以及插值、RoPE scaling 等长度外推。
- 优化：AdamW、warmup、退火、梯度裁剪、混合精度和 FlashAttention。
- 微调：SFT 数据与交叉熵；LoRA 低秩增量；Prompt/Prefix、Adapter、IA3、QLoRA 等 PEFT。
- 对齐：偏好数据、Reward Model、PPO/GRPO、reference KL 与 DPO。

SFT 学示范答案，DPO 学同一 prompt 下 chosen 相对 rejected 的偏好。训练心得应包含数据检查、模板、mask、学习率、loss/margin、过拟合和通用能力回归。

### 基础机器学习

L1 产生稀疏倾向，L2 平滑收缩权重；分类使用交叉熵是因为它来自 Bernoulli/Categorical 最大似然，配合 sigmoid/softmax 梯度合适，MSE 在概率饱和区梯度通常较弱。AUC 衡量正样本排在负样本前的概率，不等于校准或业务阈值效果。

### BERT 与论文

BERT 输入为 token、segment、position embedding，使用 MLM 与原始 NSP 预训练。需了解 RoBERTa、ALBERT、DeBERTa、ELECTRA、DistilBERT 等变体分别改了数据、参数共享、相对位置、预训练目标或蒸馏。

“最近论文”应选真正读过且与岗位相关的一篇，能讲问题、方法、公式、实验、局限和项目启发。

### 强化学习

掌握 MDP、value/advantage、policy gradient、on/off-policy、Actor-Critic、PPO clipped objective、GAE 和探索。最后连接到 LLM 对齐，但要区分传统环境逐步奖励与语言模型序列级偏好。
