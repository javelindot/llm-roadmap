### Prompt 模板库

#### 代码生成模板

```markdown
# 代码生成模板
角色: [技术栈] 专家
任务: [描述需求]
要求:
- [技术要求1]
- [技术要求2]
- [风格要求]
上下文: [参考代码或文件]
```

#### 代码审查模板

```markdown
# 代码审查模板
请审查以下代码:
- 逻辑正确性
- 安全漏洞
- 性能问题
- 代码规范
[粘贴代码]
```

### 配置模板

#### .cursorrules

```markdown
# .cursorrules 示例

## 代码风格
- 2空格缩进
- 单引号
- 分号结尾

## 组件规范
- 函数组件
- TypeScript
- Tailwind CSS
```

#### CLAUDE.md

```markdown
# CLAUDE.md 示例

## 项目
- 名称: My Project
- 技术栈: Next.js, TypeScript, Prisma

## 规范
- TypeScript strict
- ESLint + Prettier

## 命令
- 开发: npm run dev
- 构建: npm run build
```
