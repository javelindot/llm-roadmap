# Vibe Coding 学习研究教程

> 用自然语言表达意图，让 AI Agent 完成代码实现

## 什么是 Vibe Coding？

Vibe Coding 是由 Andrej Karpathy 于 2025 年提出的全新编程范式。它代表了一种核心理念的转变：**用自然语言"表达意图"，而不是用代码逐行编写实现细节**。

```
传统编程:  人类 → 编程语言 → 计算机
Vibe Coding: 人类 → 自然语言 → AI Agent → 编程语言 → 计算机
```

## 学习目标

- 理解 Vibe Coding 的本质和 Agent 架构
- 掌握 AI 编程工具的全景生态
- 深入学习 Cursor 和 Claude 两大核心范式
- 学会根据场景选择合适的工具
- 掌握 Prompt 工程与高效代码生成
- 理解企业级部署、安全与合规

## 学习建议

1. **循序渐进**：按照章节顺序学习，从基础概念到高级应用
2. **理论与实践结合**：不仅要理解理论概念，还要动手实践
3. **项目驱动**：通过实际项目加深理解和技术掌握
4. **持续学习**：AI 编程领域发展迅速，需要不断学习新技术
