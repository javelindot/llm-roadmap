# Vibe Coding 学习研究教程

> 用自然语言表达意图，让 AI Agent 完成代码实现

## 什么是 Vibe Coding？

Vibe Coding 是由 Andrej Karpathy 于 2025 年提出的全新编程范式。它代表了一种核心理念的转变：**用自然语言"表达意图"，而不是用代码逐行编写实现细节**。

```
传统编程:  人类 → 编程语言 → 计算机
Vibe Coding: 人类 → 自然语言 → AI Agent → 编程语言 → 计算机
```

## 核心理念

1. **AI 是伙伴，不是工具**：学会与 AI 协作，而不是单向发号施令
2. **表达意图比实现细节更重要**：专注于"要什么"，让 AI 处理"怎么做"
3. **验证始终必要**：AI 生成的代码需要人工审查和测试
4. **持续迭代**：通过多轮对话逐步完善代码
