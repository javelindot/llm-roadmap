# 大模型实战工具

## 目录简介

本目录介绍大模型应用开发的常用工具，包括本地部署、低代码平台、AI编程助手等。

- API接口调用

### 第二章：Dify AI 低代码平台
- Dify平台介绍
- 工作流编排

### 第三章：Coze 扣子平台

## 学习建议

1. **动手实践**：每个工具都要亲自安装使用
2. **对比选择**：根据项目需求选择合适工具
3. **关注更新**：AI工具更新快，保持学习
