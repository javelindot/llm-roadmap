# Roadmap 章节规划

基于README中的11个学习层级，收集以下学习资源链接：

## 01. 大模型应用实战 [入口核心]
- [Awesome LLM Apps](https://github.com/Shubhamsaboo/awesome-llm-apps) - 包含100+开源实现的LLM应用集合，涵盖RAG和Agent技术，支持OpenAI、Anthropic、Gemini和开源模型
- [AgentScope](https://github.com/agentscope-ai/agentscope) - 阿里巴巴通义实验室推出的生产级智能体开发框架，解决智能体在构建、运行和管理中的难题
- [Firecrawl](https://github.com/mendableai/firecrawl) - 将整个网站内容转化为适用于大语言模型的Markdown或结构化数据，通过单一API实现网页的抓取、爬行和数据提取
- [llm](https://github.com/simonw/llm) - 专为开发者和效率控打造的命令行LLM工具，将大模型真正拉进生产环境
- [MiniMax-M2.1](https://github.com/MiniMax-AI/MiniMax-M2.1) - 国产大模型，支持多语言和App开发，具有3D手势交互动画效果

## 02. 大模型实战工具
- [Ollama](https://github.com/ollama/ollama) - 开源跨平台大模型工具，在GitHub上有163K stars，是本地大模型部署领域的佼佼者，支持运行Qwen、DeepSeek-R1等多种语言模型
- [Dify](https://github.com/langgenius/dify) - 开源的LLM应用开发平台，直观的界面结合了AI工作流、RAG管道、Agent、模型管理等功能
- [MTools](https://github.com/HG-ha/MTools) - 集成了图片处理、音视频编辑、AI智能工具和开发辅助的现代化桌面工具箱，原生支持GPU加速，开箱即用
- [ERNIE Bot SDK](https://github.com/PaddlePaddle/ERNIE-Bot-SDK) - 百度ERNIE Bot的官方SDK，提供了便捷的大模型调用接口
- [Qwen SDK](https://github.com/QwenLM/Qwen) - 阿里通义千问的官方SDK，支持多语言和多模态交互

## 03. 创新应用落地
- [build-your-own-x](https://github.com/codecrafters-io/build-your-own-x) - 有47万+星标的编程学习资源库，核心理念是"通过重建你喜欢的技术来掌握编程"，收集了各种"从零造轮子"的教程
- [LLM Examples](https://github.com/openai/chatgpt-retrieval-plugin) - OpenAI官方出品的"可插拔"检索增强插件，帮助开发者在自定义数据源上实现ChatGPT问答，通过自然语言从文件、笔记、邮件和公共文档等数据源检索信息
- [ChatGLM3-6B](https://github.com/THUDM/ChatGLM3) - 清华智谱研发并开源的高性能中英双语对话语言模型，采用创新的GLM架构
- [HuaTuo](https://github.com/FudanNLP/HuaTuo) - 复旦大学研发的医疗大模型，专注于医疗健康领域的应用

## 04. NLP与预训练核心
- [Transformers](https://github.com/huggingface/transformers) - NLP领域最火的开源库之一，收录了BERT、GPT、T5、RoBERTa等数百个预训练模型，支持PyTorch与TensorFlow
- [BERT](https://github.com/google-research/bert) - 谷歌研究团队发布的双向编码器表示模型，在GitHub上获得了广泛关注，是NLP领域的重要预训练模型
- [GPT-2](https://github.com/openai/gpt-2) - OpenAI在2019年发布的生成式预训练Transformer模型，当时15亿参数被视为"危险武器"，曾因担心被用于假新闻而暂缓开源
- [PaddleNLP](https://github.com/PaddlePaddle/PaddleNLP) - 百度飞桨的NLP库，聚合了众多百度自研SOTA算法以及社区开源模型
- [awesome-pretrained-chinese-nlp-models](https://github.com/lonePatient/awesome-pretrained-chinese-nlp-models) - 中文预训练模型资源集合

## 05. 强化学习与模型对齐
- [RLHF-V](https://github.com/RLHF-V/RLHF-V.github.io) - 用于通过细粒度的纠正性人类反馈来对齐多模态大型语言模型(MLLMs)行为的框架
- [PPO Implementation](https://github.com/openai/openai-python) - OpenAI的Python客户端库，用于访问OpenAI API，在GitHub上已获得23,456个Star
- [DPO](https://github.com/eric-mitchell/direct-preference-optimization) - 直接偏好优化实现，是一种比RLHF更简单、更高效的对齐方法，不需要训练奖励模型和使用强化学习，而是直接优化LLM
- [OpenRLHF](https://github.com/OpenRLHF/OpenRLHF) - 国产开源的RLHF框架，用于大模型的对齐训练
- [AReaL](https://github.com/AReaL-Project/AReaL) - 专为大模型推理和AI Agent打造的轻量级强化学习库

## 06. 模型微调与定制
- [PEFT](https://github.com/huggingface/peft) - Hugging Face的参数高效微调库，支持LoRA、QLoRA、DoRA等多种微调方法，可在资源有限的设备上实现高效微调
- [LoRA](https://github.com/microsoft/LoRA) - 微软提出的低秩适应微调方法，通过冻结预训练权重矩阵，只训练A和B两个低秩矩阵，叠加计算结果得到最终输出
- [QLoRA](https://github.com/artidoro/qlora) - 量化低秩适应微调方法，通过将transformer模型量化为4位精度并使用分页优化器来处理内存峰值，大幅降低内存需求
- [LlamaFactory](https://github.com/hiyouga/LlamaFactory) - 国产开源的大模型统一微调框架，支持100多个LLM的微调，涵盖多种微调方法
- [SuanJia](https://github.com/suanjia-ai/suanjia) - 贵阳贵安通用基础数据微调大模型，免费开源

## 07. 多模态技术
- [CLIP](https://github.com/openai/CLIP) - OpenAI开发的对比语言-图像预训练模型，使用对比学习方法，能够同时理解图像和文本，并在两者之间建立联系，实现图像和文本之间的检索
- [m2clip](https://github.com/sallymmx/m2clip) - 利用CLIP的多任务多模态视频行为识别方法，发表于AAAI 2024 Oral
- [VLM-learning](https://github.com/WatchTower-Liu/VLM-learning) - 视觉语言模型学习资源，包含对Qwen模型的前向传播代码重写，通过中间投影层将视觉特征与文本映射到同一空间
- [Yuan3.0 Ultra](https://github.com/YuanLab-AI/Yuan) - 源3.0系列面向万亿参数规模打造的旗舰多模态基础大模型，是当前业界仅有的三个万亿级开源多模态大模型之一
- [Skywork UniPic](https://github.com/SkyworkAI/Skywork-UniPic) - 昆仑万维开源的多模态统一模型，实现图像理解、文本到图像生成、图像编辑三大核心能力
- [Skywork R1V](https://github.com/SkyworkAI/Skywork-R1V) - 昆仑万维开源的全球首个工业级多模态推理模型
- [DeepSeek-OCR2](https://github.com/deepseek-ai/DeepSeek-OCR) - 国产OCR模型，GitHub Star破5k，专注于文档处理

## 08. 机器学习基础
- [scikit-learn](https://github.com/scikit-learn/scikit-learn) - 流行的Python机器学习库，提供了丰富的工具和算法，用于数据挖掘和数据分析，建立在NumPy、SciPy和matplotlib等科学计算库的基础上
- [Awesome Machine Learning](https://github.com/josephmisiti/awesome-machine-learning) - 社区维护的机器学习与人工智能领域优秀资源集合，按语言和应用领域分类，在GitHub上有68.5K星标
- [Python Machine Learning](https://github.com/rasbt/python-machine-learning-book) - Python机器学习书籍的配套代码，以Jupyter笔记本的形式提供，包含经典机器学习教程的代码实现

## 09. 深度学习进阶
- [PyTorch](https://github.com/pytorch/pytorch) - Facebook开发的深度学习框架，提供了张量计算和自动微分功能，支持动态计算图，被广泛应用于研究和生产环境
- [TensorFlow](https://github.com/tensorflow/tensorflow) - Google开发的端到端开源机器学习平台，适用于所有人的开源机器学习框架，在GitHub上有180K+ Stars
- [Deep Learning Models](https://github.com/rasbt/deep-learning-book) - 《deep learning》书籍的代码实现，帮助读者通过Python代码理解书中的概念和算法
- [PaddlePaddle](https://github.com/PaddlePaddle/Paddle) - 百度开源的深度学习框架，支持机器学习和深度学习任务
- [MindSpore](https://github.com/mindspore-ai/mindspore) - 华为开源的全场景AI计算框架，支持机器学习和深度学习
- [OneFlow](https://github.com/Oneflow-Inc/oneflow) - 一流科技开源的深度学习框架，专注于性能优化
- [MegEngine](https://github.com/MegEngine/MegEngine) - 旷视科技开源的深度学习框架，支持高效训练和推理

## 10. 算法与系统底层
- [TensorFlow Model Optimization](https://github.com/tensorflow/model-optimization) - 谷歌官方开源的模型优化技术包，用于优化ML模型的部署，包括量化和剪枝两种API
- [PyTorch Quantization](https://github.com/pytorch/pytorch/tree/master/torch/quantization) - PyTorch中的量化工具，用于优化模型性能和减小模型大小
- [ONNX Runtime](https://github.com/microsoft/onnxruntime) - 微软开发的高性能推理引擎，支持跨平台运行ONNX格式的深度学习模型，支持CPU/GPU加速，可实现更快的客户体验和更低的成本

## 11. 附录资料与工具集
- [Designing Machine Learning Systems](https://github.com/chiphuyen/ml-interviews-book) - 机器学习系统设计指南，由Chip Huyen所著，提供了机器学习系统设计的权威参考
- [Awesome AI](https://github.com/owainlewis/awesome-artificial-intelligence) - 包含人工智能(AI)课程、书籍、视频讲座和论文等资源的集合
