## 8.1 常用工具
### 8.1.1 NLTK
NLTK（Natural Language Toolkit）是Python中最流行的自然语言处理库之一。它提供了丰富的文本处理工具，包括分词、词干提取、词性标注、命名实体识别、语义分析等功能。NLTK还包含了大量的语料库和词典资源，是NLP学习和研究的理想工具。

官方网站：[https://www.nltk.org/index.html](https://www.nltk.org/index.html)

### 8.1.2 tqdm
tqdm是一个Python进度条工具，可以轻松为循环添加进度条。它具有轻量级、易用、支持嵌套等优点，广泛应用于深度学习训练过程中显示训练进度。

官方网站：[https://tqdm.github.io/](https://tqdm.github.io/)

### 8.1.3 TensorBoard
TensorBoard是TensorFlow的可视化工具，可用于展示训练过程中的各种指标（如损失、准确率）、模型结构、数据样本等功能。它通过Web界面的形式提供交互式展示，帮助开发者直观地监控和分析模型训练过程。

使用示例：

```python
from torch.utils.tensorboard import SummaryWriter

# 创建写入器
writer = SummaryWriter(log_dir='runs/experiment_1')

# 记录标量值
writer.add_scalar('Loss/train', loss, epoch)

# 记录图像
writer.add_image('Input images', images, epoch)

# 关闭写入器
writer.close()
```

### 8.1.4 Hugging Face Transformers
Hugging Face Transformers是目前最流行的预训练模型库之一，提供了BERT、GPT、T5、LLaMA等众多预训练模型，以及易用的API接口。它极大降低了使用预训练模型的门槛，是当前NLP领域不可或缺的重要工具。

官方网站：[https://huggingface.co/docs/transformers/](https://huggingface.co/docs/transformers/)

主要功能：

+ 预训练模型加载与使用
+ 模型微调
+ 文本生成
+ 特征提取
+ 知识蒸馏

## 8.2 环境配置
### 8.2.1 Python环境
建议使用Python 3.8及以上版本。可通过conda或virtualenv创建独立的Python环境：

```bash
# 使用conda创建环境
conda create -n nlp python=3.10
conda activate nlp

# 或使用virtualenv
python -m venv nlp_env
source nlp_env/bin/activate  # Linux/Mac
nlp_env\Scripts\activate     # Windows
```

## 8.3 常见问题
### 8.3.1 GPU加速
确认CUDA版本与PyTorch兼容：

```python
import torch
print(torch.cuda.is_available())  # 检查GPU是否可用
print(torch.cuda.get_device_name(0))  # 获取GPU名称
```

### 8.3.2 内存优化
+ 使用梯度累积处理大batch
+ 及时释放不需要的变量
+ 使用混合精度训练（FP16）

### 8.3.3 模型持久化
```python
# 保存整个模型
torch.save(model, 'model.pt')

# 只保存参数（推荐）
torch.save(model.state_dict(), 'model_weights.pt')

# 加载模型
model.load_state_dict(torch.load('model_weights.pt'))
```
