# 附录资料与工具集

> 建立持续学习的资源体系

## 目录简介

本目录汇总大模型学习相关的数据集、模型资源和开发工具，为实践项目提供支持。

## 章节结构

### 第一章：数据集资源
- 文本数据集
- 多模态数据集
- 指令微调数据集
- 评估基准数据集

### 第二章：模型资源
- 开源大模型列表
- 模型下载与部署
- 模型量化与优化
- 国产大模型资源

### 第三章：开发工具
- 训练框架：PyTorch、TensorFlow、JAX
- 分布式训练：DeepSpeed、FSDP
- 模型部署：vLLM、TensorRT-LLM
- 开发环境：Jupyter、VS Code

## 使用建议

1. **按需选择**：根据项目需求选择合适的资源
2. **关注许可**：注意数据集和模型的使用许可
3. **社区贡献**：积极参与开源社区，分享资源
