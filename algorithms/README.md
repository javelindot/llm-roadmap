# 算法与系统底层

> 补全算法、编程与工程的底层能力

## 目录简介

本目录涵盖AI工程师必须掌握的算法基础、数据结构和系统设计知识。

## 章节结构

### 第一章：算法基础
- 时间复杂度与空间复杂度
- 常用算法思想
- 算法分析技巧

### 第二章：数据结构
- 线性结构：数组、链表、栈、队列
- 树形结构：二叉树、平衡树、堆
- 图结构：图的表示与遍历
- 哈希表与集合

### 第三章：系统设计
- 系统设计基础
- 分布式系统原理
- 微服务架构
- 性能优化策略

## 学习建议

1. **理论与实践结合**：理解算法原理后，通过刷题巩固
2. **关注复杂度**：始终关注时间和空间复杂度
3. **系统设计思维**：培养从整体角度思考系统的能力

## 前置知识

- 至少一门编程语言（Python/Java/C++）
- 基本的数学知识
