按以下顺序推进，每章节预计投入 1–2 天：

| 阶段 | 核心能力 | 产出 |
|------|----------|------|
| 1. Prompt 工程 | 结构化提示词、CoT、Few-shot | 能产出稳定输出的提示词库 |
| 2. Function Call | 工具调用、JSON Schema | 能让模型调用外部 API |
| 3. RAG | 向量检索、上下文增强 | 能查私有知识库的问答应用 |
| 4. MCP | 跨进程上下文协议 | 插件化的工具生态 |
| 5. Agent | 规划、记忆、多步推理 | 能自主完成复杂任务的 Agent |
