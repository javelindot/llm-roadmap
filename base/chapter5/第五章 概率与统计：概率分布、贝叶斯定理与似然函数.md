# 第五章 概率与统计：概率分布、贝叶斯定理与似然函数

## 一、概率分布

### 1.1 概率的基本概念

#### 1.1.1 概率的定义
概率是对随机事件发生可能性的度量，取值范围为$[0, 1]$。

- **古典概率**：如果样本空间$S$包含$n$个等可能的基本事件，事件$A$包含$k$个基本事件，则$P(A) = \frac{k}{n}$
- **几何概率**：对于连续型样本空间，事件$A$的概率等于其测度与样本空间测度的比值
- **统计概率**：通过频率估计概率，当试验次数足够多时，事件的频率趋近于其概率

#### 1.1.2 概率的基本性质
- 非负性：$P(A) \geq 0$
- 规范性：$P(S) = 1$（$S$为样本空间）
- 可加性：对于互不相容的事件$A_1, A_2, \dots$，有$P(\bigcup_{i=1}^\infty A_i) = \sum_{i=1}^\infty P(A_i)$

### 1.2 离散型概率分布

#### 1.2.1 伯努利分布（Bernoulli Distribution）
- **定义**：单次试验只有两种结果（成功或失败）的概率分布
- **概率质量函数（PMF）**：$P(X=k) = p^k(1-p)^{1-k}$，其中$k=0,1$，$p$为成功概率
- **期望**：$E[X] = p$
- **方差**：$Var(X) = p(1-p)$

#### 1.2.2 二项分布（Binomial Distribution）
- **定义**：$n$次独立伯努利试验中成功次数的概率分布
- **PMF**：$P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$，其中$k=0,1,\dots,n$
- **期望**：$E[X] = np$
- **方差**：$Var(X) = np(1-p)$

#### 1.2.3 泊松分布（Poisson Distribution）
- **定义**：描述单位时间内随机事件发生次数的概率分布
- **PMF**：$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$，其中$k=0,1,2,\dots$，$\lambda$为事件发生的平均速率
- **期望**：$E[X] = \lambda$
- **方差**：$Var(X) = \lambda$

#### 1.2.4 几何分布（Geometric Distribution）
- **定义**：首次成功所需试验次数的概率分布
- **PMF**：$P(X=k) = (1-p)^{k-1}p$，其中$k=1,2,\dots$
- **期望**：$E[X] = \frac{1}{p}$
- **方差**：$Var(X) = \frac{1-p}{p^2}$

### 1.3 连续型概率分布

#### 1.3.1 均匀分布（Uniform Distribution）
- **定义**：在区间$[a, b]$上概率密度恒定的分布
- **概率密度函数（PDF）**：$f(x) = \frac{1}{b-a}$，其中$a \leq x \leq b$
- **期望**：$E[X] = \frac{a+b}{2}$
- **方差**：$Var(X) = \frac{(b-a)^2}{12}$

#### 1.3.2 正态分布（Normal Distribution）
- **定义**：最常见的连续型概率分布，也称为高斯分布
- **PDF**：$f(x) = \frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$，其中$\mu$为均值，$\sigma^2$为方差
- **标准正态分布**：$\mu=0, \sigma=1$的正态分布，记为$N(0,1)$
- **期望**：$E[X] = \mu$
- **方差**：$Var(X) = \sigma^2$

#### 1.3.3 指数分布（Exponential Distribution）
- **定义**：描述事件发生时间间隔的概率分布
- **PDF**：$f(x) = \lambda e^{-\lambda x}$，其中$x \geq 0$，$\lambda > 0$
- **期望**：$E[X] = \frac{1}{\lambda}$
- **方差**：$Var(X) = \frac{1}{\lambda^2}$

#### 1.3.4 伽马分布（Gamma Distribution）
- **定义**：指数分布的推广，用于描述多个独立同分布指数变量的和
- **PDF**：$f(x) = \frac{\lambda^\alpha x^{\alpha-1} e^{-\lambda x}}{\Gamma(\alpha)}$，其中$x \geq 0$，$\alpha > 0$，$\lambda > 0$
- **期望**：$E[X] = \frac{\alpha}{\lambda}$
- **方差**：$Var(X) = \frac{\alpha}{\lambda^2}$

### 1.4 多维概率分布

#### 1.4.1 联合分布
- **离散型**：联合概率质量函数$P(X=x, Y=y)$
- **连续型**：联合概率密度函数$f(x, y)$

#### 1.4.2 边缘分布
- 从联合分布中提取单个变量的分布
- 离散型：$P(X=x) = \sum_y P(X=x, Y=y)$
- 连续型：$f_X(x) = \int_{-\infty}^{\infty} f(x, y) dy$

#### 1.4.3 条件分布
- 在已知一个变量取值的条件下，另一个变量的分布
- 离散型：$P(X=x | Y=y) = \frac{P(X=x, Y=y)}{P(Y=y)}$
- 连续型：$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$

## 二、贝叶斯定理

### 2.1 贝叶斯定理的基本形式
贝叶斯定理描述了如何根据新证据更新先验概率：

$$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$$

其中：
- $P(A|B)$：后验概率，在证据$B$下$A$的概率
- $P(B|A)$：似然度，$A$发生时$B$的概率
- $P(A)$：先验概率，$A$的初始概率
- $P(B)$：证据的边际概率

### 2.2 贝叶斯定理的应用

#### 2.2.1 医学诊断
- **问题**：已知某种疾病的患病率$P(D)$，检测的真阳性率$P(+|D)$和假阳性率$P(+|\neg D)$，求检测阳性时患病的概率$P(D|+)$
- **计算**：$P(D|+) = \frac{P(+|D)P(D)}{P(+|D)P(D) + P(+|\neg D)P(\neg D)}$

#### 2.2.2 垃圾邮件过滤
- **问题**：根据邮件中的关键词判断是否为垃圾邮件
- **应用**：计算在出现特定关键词的情况下，邮件是垃圾邮件的概率

#### 2.2.3 机器学习中的贝叶斯方法
- **朴素贝叶斯分类器**：假设特征之间条件独立，利用贝叶斯定理进行分类
- **贝叶斯网络**：通过有向无环图表示变量之间的条件依赖关系
- **高斯过程**：用于回归和分类的非参数贝叶斯方法

### 2.3 贝叶斯定理的扩展

#### 2.3.1 全概率公式
对于互斥且穷尽的事件$A_1, A_2, \dots, A_n$，有：

$$P(B) = \sum_{i=1}^n P(B|A_i)P(A_i)$$

#### 2.3.2 贝叶斯更新
随着新证据的获得，不断更新后验概率：
- 初始先验：$P(A)$
- 获得证据$B_1$后：$P(A|B_1) = \frac{P(B_1|A)P(A)}{P(B_1)}$
- 获得证据$B_2$后：$P(A|B_1, B_2) = \frac{P(B_2|A, B_1)P(A|B_1)}{P(B_2|B_1)}$

### 2.4 贝叶斯定理与频率学派的区别
- **频率学派**：概率是事件发生的相对频率，参数是固定的未知常数
- **贝叶斯学派**：概率是对事件发生可能性的主观信念，参数是随机变量，具有先验分布

## 三、似然函数

### 3.1 似然函数的定义
似然函数是参数的函数，表示在给定参数下观察到当前数据的概率：

$$L(\theta | X) = P(X | \theta)$$

其中：
- $X$：观察到的数据
- $\theta$：模型参数
- $L(\theta | X)$：在参数$\theta$下，观察到数据$X$的概率

### 3.2 似然函数与概率的区别
- **概率**：固定参数，预测数据的分布
- **似然**：固定数据，评估不同参数的可能性

### 3.3 最大似然估计（Maximum Likelihood Estimation, MLE）

#### 3.3.1 基本思想
选择使似然函数最大化的参数值，即：

$$\hat{\theta}_{MLE} = \arg\max_{\theta} L(\theta | X)$$

由于对数函数是单调递增的，通常最大化对数似然函数：

$$\hat{\theta}_{MLE} = \arg\max_{\theta} \ln L(\theta | X)$$

#### 3.3.2 示例：正态分布的MLE
对于正态分布$N(\mu, \sigma^2)$，样本$x_1, x_2, \dots, x_n$的对数似然函数为：

$$\ln L(\mu, \sigma^2 | X) = -\frac{n}{2}\ln(2\pi) - \frac{n}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^n (x_i - \mu)^2$$

通过求导并令其为零，得到：
- $\hat{\mu}_{MLE} = \frac{1}{n}\sum_{i=1}^n x_i$（样本均值）
- $\hat{\sigma}^2_{MLE} = \frac{1}{n}\sum_{i=1}^n (x_i - \hat{\mu})^2$（样本方差）

#### 3.3.3 示例：二项分布的MLE
对于二项分布$Bin(n, p)$，成功次数为$k$，对数似然函数为：

$$\ln L(p | k) = \ln\binom{n}{k} + k\ln p + (n-k)\ln(1-p)$$

求导并令其为零，得到：
- $\hat{p}_{MLE} = \frac{k}{n}$（成功次数的比例）

### 3.4 似然函数的性质
- **可加性**：对于独立同分布的样本，似然函数是各个样本似然的乘积
- **不变性**：如果$\hat{\theta}$是$\theta$的MLE，那么$g(\hat{\theta})$是$g(\theta)$的MLE（$g$为单调函数）
- **渐近性质**：随着样本量增加，MLE趋近于真实参数，且服从正态分布

### 3.5 似然比检验
似然比检验用于比较两个嵌套模型的拟合优度：

$$\Lambda = \frac{L(\theta_0 | X)}{L(\hat{\theta} | X)}$$

其中：
- $\theta_0$：原假设下的参数值
- $\hat{\theta}$：无约束的MLE
- $-2\ln\Lambda$：服从卡方分布，可用于假设检验

### 3.6 似然函数在机器学习中的应用
- **参数估计**：通过最大化似然函数估计模型参数
- **模型选择**：使用AIC、BIC等基于似然的准则选择模型
- **贝叶斯推断**：似然函数与先验分布结合，得到后验分布
- **EM算法**：处理含有隐变量的模型的参数估计
