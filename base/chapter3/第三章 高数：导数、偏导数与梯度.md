# 第三章 高数：导数、偏导数与梯度

## 一、导数

### 1.1 导数的定义
导数是函数在某一点的变化率，数学定义为：

$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

### 1.2 基本导数公式
- 常数函数：$(C)' = 0$（$C$为常数）
- 幂函数：$(x^n)' = nx^{n-1}$（$n$为实数）
- 指数函数：$(e^x)' = e^x$，$(a^x)' = a^x \ln a$（$a > 0$且$a \neq 1$）
- 对数函数：$(\ln x)' = \frac{1}{x}$，$(\log_a x)' = \frac{1}{x \ln a}$（$a > 0$且$a \neq 1$）
- 三角函数：
  - $(\sin x)' = \cos x$
  - $(\cos x)' = -\sin x$
  - $(\tan x)' = \sec^2 x$
  - $(\cot x)' = -\csc^2 x$

### 1.3 导数的运算法则
- 加法法则：$(u + v)' = u' + v'$
- 乘法法则：$(uv)' = u'v + uv'$
- 除法法则：$(\frac{u}{v})' = \frac{u'v - uv'}{v^2}$（$v \neq 0$）
- 复合函数求导（链式法则）：$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$

### 1.4 导数的几何意义
导数表示函数图像在某一点的切线斜率，即：
- $f'(x_0)$是曲线$y = f(x)$在点$(x_0, f(x_0))$处的切线斜率
- 切线方程：$y - f(x_0) = f'(x_0)(x - x_0)$

## 二、偏导数

### 2.1 偏导数的定义
对于多元函数$z = f(x, y)$，偏导数定义为：

- 对$x$的偏导数：$f_x(x, y) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x, y) - f(x, y)}{\Delta x}$
- 对$y$的偏导数：$f_y(x, y) = \lim_{\Delta y \to 0} \frac{f(x, y + \Delta y) - f(x, y)}{\Delta y}$

### 2.2 偏导数的计算
计算偏导数时，将其他变量视为常数，对目标变量求导：

**示例**：
- 对于$f(x, y) = x^2 + 3xy + y^2$，
  - $f_x = 2x + 3y$（将$y$视为常数）
  - $f_y = 3x + 2y$（将$x$视为常数）

### 2.3 高阶偏导数
二阶偏导数包括：
- $f_{xx} = \frac{\partial^2 f}{\partial x^2}$（对$x$求二阶偏导）
- $f_{xy} = \frac{\partial^2 f}{\partial x\partial y}$（先对$x$求偏导，再对$y$求偏导）
- $f_{yx} = \frac{\partial^2 f}{\partial y\partial x}$（先对$y$求偏导，再对$x$求偏导）
- $f_{yy} = \frac{\partial^2 f}{\partial y^2}$（对$y$求二阶偏导）

**注意**：当二阶混合偏导数连续时，$f_{xy} = f_{yx}$（Clairaut定理）

## 三、梯度

### 3.1 梯度的定义
对于多元函数$f(x_1, x_2, \dots, x_n)$，梯度是一个向量，定义为：

$$\nabla f = \left( \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \dots, \frac{\partial f}{\partial x_n} \right)$$

### 3.2 梯度的几何意义
- 梯度向量的方向是函数值增长最快的方向
- 梯度向量的模长是函数在该方向上的最大变化率
- 梯度向量与函数的等值面（或等值线）正交

### 3.3 梯度在机器学习中的应用
- **梯度下降法**：通过沿着负梯度方向更新参数，最小化损失函数
- **反向传播算法**：利用梯度计算神经网络中各层参数的偏导数
- **优化算法**：如Adam、SGD等都基于梯度信息进行参数更新

### 3.4 梯度的计算
**示例**：
- 对于$f(x, y) = x^2 + y^2$，
  - 梯度$\nabla f = (2x, 2y)$
  - 在点$(1, 1)$处，梯度为$(2, 2)$，表示函数在该点沿$(2, 2)$方向增长最快

### 3.5 梯度的性质
- 线性性：$\nabla(f + g) = \nabla f + \nabla g$，$\nabla(cf) = c\nabla f$（$c$为常数）
- 乘积法则：$\nabla(fg) = f\nabla g + g\nabla f$
- 链式法则：对于复合函数$h(x) = f(g(x))$，$\nabla h(x) = (\nabla f)(g(x)) \cdot \nabla g(x)$
